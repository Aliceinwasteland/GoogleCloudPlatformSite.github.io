Google App Engine

App Engine is a platform as a service that uses familiar technologies to build and host applications on the same infrastructure used at Google.

Learn More
Features:
Platform as a service

You deploy your application to Google’s infrastructure; we wear the pagers. App Engine automatically takes care of provisioning and balancing requests across instances and data centers. So you can focus on writing great applications.

Popular programming language support

Write applications using any of App Engine’s supported languages including Java, Python, Go and PHP. Each has its own runtime dedicated to running your code fast and securely.

Flexible and scalable application storage

App Engine provides a range of options to store your data. The App Engine Datastore is schemaless and is used to store non-relational data. Google Cloud SQL provides a relational SQL database based on MySQL. Google Cloud Storage provides storage for objects and files up to terabytes in size.

Memcache

Memcache gives your application a high performance in-memory key-value cache that is accessible by multiple instances of your application. Memcache is useful for data that does not need the persistence and transactional features of the datastore, such as temporary data or data copied from the datastore to the cache for high speed access.

Task queues

With task queues, you can easily defer work from user-facing requests for later processing. This simplifies application logic and improves latency.

Auto-scaling

Automatic scaling is built in with App Engine, all you have to do is write your application code and we'll do the rest. No matter how many users you have or how much data your application stores, App Engine can scale to meet your needs.

Local developer tools

With the App Engine SDK, you can develop, test and iterate your applications locally without doing full deployments. App Engine also offers integration with multiple IDE’s, including the Google Plugin for Eclipse.

Third party frameworks and extensions

Take advantage of the frameworks and libraries that are compatible with App Engine. For Python, this includes webapp2, Django, Flask and Tornado; for Java, you can use Spring; for PHP, App Engine offers deep integration with Wordpress.

Google Compute Engine

Compute Engine is an infrastructure as a service that lets you run your large-scale computing workloads on Linux virtual machines hosted on Google's infrastructure.

Learn More
Features:
Infrastructure as a service

Power your existing and new applications using Linux virtual machines in Google’s data centers. Choose the right size container for your workload from micro instances priced less than two pennies an hour to large powerful instances available in high-memory and high-CPU configurations.

High Data Security

All data written to Google Compute Engine’s disk storage options is encrypted using the AES-128-CBC algorithm at rest.

Connect with the Google Network

Create large compute clusters that benefit from strong and consistent cross-machine bandwidth. Connect to machines in other data centers and to other Google services using Google’s fast private global network.

Fast and Easy Provisioning

Get up and running quickly with intuitive and easy to use tools. Quickly deploy large clusters of virtual machines.

Flexible Billing

Pay for what you use. Enjoy minute-level billing increments (with a 10 minute minimum charge) so you don’t pay for compute minutes that you don’t use.

Large and Powerful Disks

Create up to 10TB disk volumes that offer strong and predictable read and write performance.

Green Computing

Use data centers that are up to 50% more efficient than traditional data centers, run by the first major internet service provider to achieve external certification of environmental standards. Benefit from the cost savings that highly optimized, internet scale facilities deliver.

Load Balancing

Native load balancing technology helps you spread ingress network traffic across a pool of instances to cost-effectively maximize application performance, throughput, and availability.

Partner Powered

Access best-of-breed management tools through deep partner integration with providers like RightScale, Scalr, Opscode and Puppet Labs.

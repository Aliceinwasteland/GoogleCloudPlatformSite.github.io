Benefícios da criação de aplicativos móveis no Google Cloud Platform:

Escalonamento e balanceamento de carga automáticos

Tudo o que você tem que fazer é escrever o código de seu aplicativo. Nós cuidaremos do resto. Independentemente de quantos usuários você tem, o Google App Engine pode ser escalonado de acordo com suas necessidades.

Fornecimento de conteúdo estático

O App Engine fornece arquivos estáticos a partir de caches e servidores dedicados, que são mantidos separadamente dos servidores de aplicativo.

Processamento de dados

Registros de aplicativos podem fornecer informações sobre o uso do aplicativo e o comportamento backend. O BigQuery permite reunir em tempo real análise ad-hoc de seus arquivos de registros, analisando terabytes de dados em segundos.

Pesquisa por proximidade geográfica

Aproveite a API de pesquisa do Google App Engine para pesquisar por proximidade geográfica.

Notificações push

Use o Google Cloud Messaging para enviar dados a usuários do Android e da API de soquetes a fim de de enviar notificações push a seus usuários de iOS.

Armazenamento de dados

O Google Cloud Storage permite armazenar objetos e arquivos de até terabytes de tamanho usando uma interface simples baseada em REST.

Soluções

Fornecer grandes arquivos de mídia

Os usuários exigem cada vez mais imagens de alta resolução e downloads de todos os tamanhos. Garanta o retorno dos usuários e seu comprometimento com o aplicativo oferecendo conteúdo sempre atualizado. O Google Cloud Storage oferece uma interface simples baseada em REST para armazenar grandes arquivos na nuvem e usa a mesma rede de alta velocidade que alimenta os aplicativos do Google.

Analisar o envolvimento dos usuários

Entender o que os usuários estão fazendo e como seu aplicativo está sendo executado é a chave para desvendar o valor de seu aplicativo. Mas, para isso, você precisa mais do que apenas um registro; você precisa de uma análise poderosa. O Google BigQuery apresenta a mesma tecnologia que usamos no Google para resolver esses problemas em seus aplicativos, permitindo executar análises ad-hoc em tempo real de terabytes de dados para alimentar seus painéis. Ou você pode usar MapReduce no Gogole App Engine para processar grandes conjuntos de dados em paralelo para análises que não sejam em tempo real.

Sincronizar estados de jogos em vários dispositivos

Armazene stados de jogos no Armazenamento de dados do Google App Engine ou no Armazenamento de dados do Google Cloud Storage, disponibilizando-os a partir de todos os dispositivos de seus usuários e forneça-os com a capacidade de obter informações detalhadas a partir deles.

Integrar facilmente com dispositivos móveis

Use o Google Cloud Endpoints para expor automaticamente a lógica de seu servidor como APIs da Web e gerar bibliotecas de cliente para iOS, Android e JavaScript para aplicativos da Web padrão.

Criar grande variedade de jogos no Google Cloud Platform

O Google Cloud Platform permite criar jogos para plataformas móveis, incluindo Android e iOS, e plataformas da Web, incluindo Chrome OS e Facebook. Você também pode criar diferentes tipos de jogos, desde jogos para serem jogados com outos jogadores até jogos sociais.

Distribuir e monetizar seu jogo com o Google

Gere receita com seu jogo usando o Google AdSense e o Google In-App Payments. O Google Adsense permite exibir anúncios segmentados que correspondam à aparência de seu jogo. O Google In-App Payments permite vender bens virtuais com uma taxa fixa de 5%. Distribua seu jogo usando a Chrome Web Store, para jogos da Web, ou o Google Play para jogos, para dispositivos móveis.

Cloud Datastore

Use um banco de dados NoSQL gerenciado e sem esquemas para armazenar dados não-relacionais. O Google Cloud Datastore é dimensionado automaticamente à medida que você precisa dele e suporta transações e consultas robustas, semelhantes a SQL.

Recursos

Acesso sem esquemas, com consultas semelhantes a SQL

Não há necessidade de se preocupar com modelos de dados e migração. O Google Cloud Datastore é um serviço de armazenamento sem esquemas que permite ser ágil, eliminando a necessidade de se preocupar com a estrutura subjacente dos dados. O Google Cloud Datastore fornece um mecanismo de consulta robusto que permite pesquisar dados em várias propriedades e classificações, conforme necessário.

Banco de dados gerenciado

O Google Cloud Datastore é totalmente gerenciado. O Google lida automaticamente com fragmentação e replicação para fornecer um banco dedados altamente disponível e consistente.

Escalabilidade automática com seus usuários

O Google Cloud Datastore é dimensionado automaticamente de acordo com suas necessidades. Isso permite que você se concentre na criação de seu aplicativo e não em se preocupar com o provisionamento e a antecipação de cargas.

Transações ACID

O Google Cloud Datastore fornece transações ACID usando controle de simultaneidade otimista. Seu aplicativo pode executar diversas operações de armazenamento de dados em uma única transação, em que todas serão executadas ou falharão, assegurando assim a integridade de seus dados.

Redundância incorporada

Com uma única chamada de API, os dados são automaticamente replicados em vários centros de dados. Alta disponibilidade e durabilidade são incorporadas desde o primeiro núcleo.

Ferramentas de desenvolvimento local

Com o Kit de desenvolvimento do Google Cloud Datastore, você pode desenvolver, testar e executar repetidamente seus aplicativos sem fazer implementações completas.

Acesse seus dados de qualquer lugar

Crie soluções que abranjam o Google App Engine e o Google Compute Engine e contem com o Google Cloud Datastore como o ponto de integração. Com a interface RESTful que é exposta pelo Google Cloud Datastore, os dados podem ser facilmente acessados ​​por qualquer destino de implementação.

Preços

As taxas e a camada gratuita do Google Cloud Datastore são as mesmas que as taxas de armazenamento de dados do Google App Engine, que são detalhadas a seguir. Você deve estar ciente de que o Google Cloud Datastore tem um componente de fornecimento que é executado no Google App Engine, de modo que não há custos por hora para as instâncias.

Recursos

  • Visão geral do Google Cloud Datastore

    O Google Cloud Datastore fornece um conjunto de ferramentas e bibliotecas cliente para acessar e gerenciar o Google Cloud Datastore de clientes que executam em instâncias do Google Cloud Compute Engine ou em outros lugares. Ele fornece uma API baseada em RPC hospedada pelo mesmo serviço do Google que hospeda o Google Maps, o YouTube e as APIs do Google Apps. Você pode usá-lo com a maioria dos aplicativos e linguagens, por exemplo, com um aplicativo da Web node.js executado em uma máquina virtual ou um servidor de aplicativos Java no local.

  • Ativar o Google Cloud Datastore

    Veja a seguir uma descrição de como ativar a API do Google Cloud Datastore, as cotas associadas a essa API e como ativar o faturamento. O processo de ativação pode variar de acordo com a configuração e o ambiente já existente.

  • Conceitos-chave do Google Cloud Datastore

    O Google Cloud Datastore é um armazenamento de dados de objetos sem esquemas escalável e robusto para seu aplicativo, com os seguintes recursos: não há tempo de inatividade não planejado, transações atômicas, alta disponibilidade de leituras e escritas, grande consistência para leituras e consultas ancestrais e consistência eventual para todas as outras consultas.

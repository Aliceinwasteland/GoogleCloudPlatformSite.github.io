Cloud SQL

Armazene e gerencie dados usando um banco de dados MySQL relacional totalmente gerenciável. O Google lida com replicação, gerenciamento de patches e gerenciamento de banco de dados para garantir disponibilidade e desempenho.

Recursos

Bancos de dados MySQL implantados na nuvem sem transtornos. O Google Cloud Platform fornece bancos de dados poderosos que são executados rapidamente, não ficam sem espaço e oferecem a seu aplicativo o armazenamento redundante e confiável que ele necessita.

Infraestrutura familiar

Crie e implante na nuvem mais rapidamente, poque o Google Cloud SQL oferece bancos de dados MySQL padrão, o banco de dados de código aberto mais popular do mundo. Instâncias disponíveis até 16 GB de RAM, 100 GB de armazenamento.

Carregamento flexível

Nossa opção de pagamento por uso torna econômico o processo inicial. Se você estiver executando um banco de dados que é pouco usado ou usado esporadicamente, economizará dinheiro, pagando somente pelo tempo que acessar seus dados. A opção de pacotes permite controlar seus custos para instâncias muito carregadas.

Segurança, disponibilidade, durabilidade

Your data is replicated in many geographic locations as standard, and failover between them is handled automatically. This means your data is safe and your database is available even in the event of a major failure. We also manage your backups, making it easy for you to restore when needed, including point-in-time recovery.

Migração mais fácil; sem bloqueio

Conexões e ferramentas padrão, como mysqldump, MySQL Wire Protocol e JDBC facilitam a migração para o Google Cloud Platform e evitam o bloqueio.

Controle

It's easy to manage and access your instances through a web Console or a command-line interface.

Totalmente gerenciado

Deixe de se preocupar com replicação, gerenciamento de patches ou de banco de dados; nós cuidamos disso.

Preços

O Google oferece dois planos de faturamento para o Google Cloud SQL:

  • Para desenvolvedores com mais tráfego, temos planos em pacotes com desconto quee ajudam a prever os seus custos com antecedência.
  • Para desenvolvedores com aplicativos mais leves, oferecemos um esquema de preços flexível "por uso". Você só paga pelo tempo que acessa seus dados. Comece a trabalhar com um banco de dados MySQL hospedado na nuvem por aproximadamente $ 1,00 por mês.

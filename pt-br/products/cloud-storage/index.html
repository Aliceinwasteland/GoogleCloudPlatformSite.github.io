Cloud Storage

Use um serviço de armazenamento de objetos durável ​​e altamente disponível. Com armazenamento em cache global, seus usuários têm rápido acesso aos dados de seu aplicativo a partir de qualquer localização.

Recursos

Seguro e protegido

Seus dados são protegidos por meio de armazenamento redundante em vários locais físicos. Controles de acesso OAuth e granular formam uma proteção forte e configurável.

Preços competitivos e flexíveis

Escolha uma solução de armazenamento que se adapte à natureza de seus dados, sejam ela de armazenamento padrão ou armazenamento de disponibilidade durável reduzida (DRA). Assim como acontece com outros serviços do Gogole Clould Platform, você só paga pelo que usa.

Armazenamento de objetos com uma API repleta de recursos

Embora o Google Cloud Storage possa ser acessado com uma API simples, conjuntos de recursos avançados acrescentam flexibilidade e poder. Controle de versão de objetos, envios paralelos e verificação de integridade baseada em CRC fornecem aos desenvolvedores a capacidade de criar aplicativos robustos e sofisticados. Você pode acessar a API usando XML ou JSON diretamente ou usar bibliotecas para várias linguagens de programação populares.

Acesso flexível

Você pode fornecer objetos estáticos diretamente do Google Cloud Storage ou usá-lo como um repositório de dados para outros produtos do Google. Acesse seu armazenamento a partir do BigQuery, do Google App Engine ou de outros serviços do Google Cloud Platform. Seus dados podem ser acessados por meio de uma API RESTful autenticada que fornece velocidade, flexibilidade e segurança suportada por OAuth.

  • Visão geral do Google Cloud Storage

    O Google Cloud Storage permite o armazenamento mundial e a recuperação de qualquer volume de dados e a qualquer momento. Ele fornece uma interface de programação simples, que permite aos desenvolvedores aproveitar a infraestrutura de rede confiável e rápida do Google para executar operações de dados de forma segura e econônica. Se surgirem necessidades de expansão, os desenvolvedores podem se beneficiar da escalabilidade fornecida pela infraestrutura do Google.

Compute

Compute Engine

O Google Compute Engine é a Infrastructure-as-a-Service (IaaS) do Google. Execute cargas de trabalho em grande escala em máquinas virtuais hospedadas na infraestrutura do Google. Escolha uma máquina virtual que se adapte a suas necessidades e obtenha o desempenho e a consistência da rede de fibra óptica mundial do Google. Com o faturamento por minuto, você paga somente pelo que usa.

Google App Engine

O Google App Engine é a Plataforma como um serviço (PaaS) do Google. Desenvolva seu aplicativo com facilidade, usando serviços integrados que tornam seu produto mais produtivo. Faça a implementação em uma plataforma totalmente gerenciável e deixe que o Google carregue o pager. Basta fazer o download do SDK e começar a criar imediata e gratuitamente, sem a necessidade de um cartão de crédito.

Armazenamento

Google Cloud SQL

Armazene e gerencie dados usando um banco de dados MySQL relacional totalmente gerenciável. O Google lida com replicação, gerenciamento de patches e gerenciamento de banco de dados para garantir disponibilidade e desempenho.

Cloud Storage

Use um serviço de armazenamento de objetos durável e altamente disponível. Com armazenamento em cache global, seus usuários têm rápido acesso aos dados de seu aplicativo de qualquer localização. O Google gerencia versões, garante um SLA robusto e fornece uma API simples que permite que você gerencie seus dados de forma programática.

Cloud Datastore

O Google Cloud Datastore fornece um banco de dados gerenciado, sem esquemas e sem SQL para o armazenamento de dados não relacionais. O Google Cloud Datastore é dimensionado automaticamente à medida que você precisa e suporta transações e consultas robustas, semelhantes a SQL.

BigQuery

Serviços

Cloud DNS

Servidor DNS confiável, resistente e de baixa latência da rede mundial de servidores Anycast DNS do Google. Criar registros de DNS com uma interface de linha de comando fácil de usar ou programar contra uma RESTful API com funcionalidade total para customizar o serviço para suas necessidades específicas.

Google Cloud Endpoints

Crie serviços RESTful de seu código e torne-os acessíveis a clientes iOS, Android e Javascript. Gere bibliotecas de clientes automaticamente para tornar a fiação até a interface fácil. A infraestrutura crítica integrada inclui proteção a negação de serviço, suporte a OAuth 2.0 e gerenciamento de chaves do cliente.

Gerenciamento

Cloud Deployment Manager

O Cloud Deployment Manager permite aos desenvolvedores criar, compartilhar, implantar e gerenciar com facilidade soluções complexas da Cloud Platform, usando modelos simples e declarativos. De servidores da Web simples a complexos clusters altamente disponíveis, o Deployment Manager permite que as equipes gastem menos tempo gerenciando e mais tempo criando.

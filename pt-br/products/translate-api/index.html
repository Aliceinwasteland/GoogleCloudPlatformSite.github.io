Translate API — Google Cloud Platform

Translate API

Traduza, com rapidez e de forma dinâmica, entre milhares de pares de idiomas disponíveis em seu aplicativo, integrando com o Google Tradutor.

Fazer o teste agora

Recursos

Acesse idiomas de forma dinâmica

A API do Google Tradutor suporta dúzias de idiomas . Detecte o idioma de um documento e traduza-o usando uma API RESTful.

Acessível com a API do Google

A API do Google Tradutor pode ser acessada por meio da API padrão do Google. Consulte código e bibliotecas de amostra para obter dez diferentes linguagens de programação, incluindo Python, Objective C e Ruby.

Preço acessível

Os preços da API do Google Tradutor baseiam-se em milhões de caracteres enviados. Nosso programa de preços é fácil de entender e muito acessível.

Preços

Os preços da API do Google Tradutor baseiam-se no uso. O uso do Google Translator é calculado em milhões de caracteres (M), em que 1 M = 10 6 caracteres. Para obter mais informações, consulte a seção Perguntas frequentes sobre preços.

Importante: A API do Google Tradutor v2 exige informações de faturamento para todas as contas antes da utilização do serviço. Veja instruções abaixo sobre como ativar o faturamento.

  • Tarifas de uso:
    • Tradução:
      • US$ 20,00 por 1 M caracteres de texto; as cobranças são ajustadas de acordo com o número de caracteres fornecidos no momento. Por exemplo, se você traduzir 500K caracteres, serão cobrados US$ 10,00.
    • Detecção de idioma:
      • US$ 20,00 por 1 M caracteres de texto; as cobranças são ajustadas de acordo com o número de caracteres fornecidos no momento.
  • Limites de uso:
    • A API do Google Tradutor tem um limite padrão de 2 M caracteres/dia. Esse limite pode ser aumentado para 50 M caracteres/dia no Cloud Console, seguindo as instruções abaixo.
    • Se precisar traduzir mais de 50 M caracteres/dia, fale conosco

Para ativar o faturamento para seu projeto, faça o seguinte:

  1. Vá para o Google Cloud Console.
  2. Selecione um projeto.
  3. Na barra lateral do lado esquerdo, selecione Faturamento.
  4. Na parte superior da página, selecione Ativar faturamento.
  5. Preencha o formulário.

Para visualizar ou alterar limites de uso para seu projeto ou para solicitar um aumento para seu limite total, faça o seguinte:

  1. Vá para o Google Cloud Console.
  2. Selecione um projeto.
  3. Na barra lateral do lado esquerdo, selecione APIs & auth e selecione uma API.
  4. Na página de informações da API, selecione o link Cota , perto do nome da API.

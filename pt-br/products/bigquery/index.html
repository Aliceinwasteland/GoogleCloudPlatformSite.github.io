BigQuery

Analise o Big Data na nuvem com o BigQuery. Execute consultas semelhantes a SQL com rapidez em conjuntos de dados de vários terabytes em segundos. Escalável e fácil de usar, o BigQuery fornece informações em tempo real sobre seus dados.

Tudo nos bastidores

Suas consultas podem ser executadas de forma assíncrona em segundo plano e podem ser pesquisadas por status. Acesse um histórico de suas tarefas e consultas junto com os demais elementos do Google Cloud Platform no Google Cloud Console.

Importe dados com facilidade

Carregue seus dados em massa usando o Google Cloud Storage ou transmita-os em bursts de até 1.00 linhas por segundo. É possível lidar em tempo real com registros em grande volume ou gravação de transações e ter seus dados disponíveis para análise com rapidez e facilidade.

Dados volumosos acessíveis

Quando dizemos "dados volumosos", é isso realmente o que queremos dizer. Os primeiros 100 GB de dados processados ​​a cada mês são gratuitos. Armazenamos e consultamos milhares de terabytes para você. Nossa estrutura de preços é simples e transparente, e damos a você visibilidade e controle totais. Você só paga pelo armazenamento necessário e pelas consultas que executar.

A interface certa

Interfaces separadas para administração e desenvolvedores garantem o acesso às ferramentas de que você precisa. Controle o acesso, tanto em nível de projeto como de conjunto de dados, usando o Consolte de APIs Google. O BigQuery ajuda equipes de qualquer tamanho a trabalhar da maneira que desejarem.

Preços

O Google BigQuery usa uma estrutura de dados em colunas, o que significa que, para uma consulta específica, você só é cobrado pelos dados processados em cada coluna, não pela tabela inteira. Os primeiros 100 GB de dados processados por mês não têm custo.

Preços válidos a partir de 1 de abril de 2014.

Recursos

  • O que é o BigQuery?

    A consulta de conjuntos de dados massivos pode ser demorada e cara, sem a infraestrutura e o hardware adequados. O Google BigQuery resolve esse problema, permitindo consultas super rápidas, semelhantes a SQL, em tabelas apenas por anexação, usando o poder de processamento de infraestrutura do Google. Basta mover seus dados para o BigQuery e lidar com o trabalho difícil.

  • Como carregar dados no BigQuery

    Para poder consultar seus dados, você primeiro precisa carregá-los no BigQuery. Você pode carregar os dados em massa usando uma tarefa ou executar registros em fluxo individualmente.

  • Como consultar dados

    As consultas são escritas no {0}dialeto do SQL do BigQuery{/0}. O BigQuery suporta métodos de consultas síncronas e assíncronas. Ambos os métodos são tratados por um trabalho, mas o método "síncrono" expõe um valor de tempo limite que aguarda até que o trabalho tenha sido concluído para retornar.

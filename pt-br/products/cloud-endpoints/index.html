Cloud Endpoints

Crie serviços RESTful e torne-os acessíveis a clientes iOS, Android e Javascript. Gere bibliotecas cliente automaticamente para tornar a fiação até a interface fácil. Recursos incorporados incluem proteção a negação de serviço, suporte a OAuth 2.0 e gerenciamento de chaves do cliente.

Uma ferramenta, vários clientes

Crie bibliotecas de clientes para Android, iOS e clientes baseados na Web de uma única fonte. O Google Cloud Endpoints envolve seu código para criar um servidor de APIs em apenas alguns passos. Bibliotecas de APIs do Google Cloud Endpoints estão disponíveis em Java, Python, Go e PHP. Saiba mais sobre o Google Cloud Endpoints em nossa documentação ou simplesmente experimente nosso aplicativo para Web de amostra, o Tic Tac Toe.

Como estender a infraestrutura do Google App Engine

Todas as ferramentas e bibliotecas disponibilizados no Google App Engine estão agora disponíveis para seus dispositivos móveis. Acesse o armazenamento de dados, o Google Cloud Storage e as filas de tarefas, usando seu back-end do Google App Engine sem nenhum trabalho extra. Integre a autenticação OAuth 2.0 rapidamente, seguindo os nossos exemplos.

Cliente-servidor de baixa manutenção

Como o Google Cloud Endpoints é apoiado pelo Google App Engine, você não tem servidores para manter, nem balanceamento de carga para se preocupar e tem o mesmo escalonamento rápido e sem esforço. Assim como acontece com o Google App Engine e com nossos outros serviços na nuvem, você só paga pelo que usa.

Integração flexível do lado do cliente

Faça a notação de sua API do lado do servidor e crie suas bibliotecas de cliente automaticamente. Bibliotecas de cliente são criadas para Android e iOS . Deixe clientes da Web prontos para funcionar com uma biblioteca de cliente JavaScript mínima. Todos os seus clientes usam APIs semelhantes e ao mesmo back-end, o que reduz o tempo de desenvolvimento.

Preços

O Google Cloud Endpoints está disponível gratuitamente para aplicativos executados no Google App Engine e no Google Compute Engine.

Documentação e recursos

Crie APIs clientes automaticamente para seus aplicativos do Google App Engine que usam o Google Cloud Endpoints. Tutoriais e instruções estão disponíveis para os vários tempos de execução do Google App Engine que ajudarão você a começar a executar rapidamente.

Recursos

  • Como criar uma API do Google Cloud Endpoints

    Uma API do Google Cloud Endpoints é um serviço de chamada de procedimento remoto (RPC) que fornece métodos remotos acessíveis a clientes externos. Cada API do Google Cloud Endpoints consiste em uma classe de serviço RPC que apresenta subclasses para o ProtoRPC remoto.Classe de serviço e um ou mais métodos.

  • Como gerar bibliotecas cliente

    Depois que terminar de criar seus pontos de extremidade, você poderá gerar bibliotecas cliente para Android e iOS

  • Como usar o Google Cloud Endpoints em um cliente iOS

    Para usar o Google Cloud Endpoints em um aplicativo iOS, você precisa: compilar o gerador da biblioteca cliente e gerar sua biblioteca; adicionar arquivos necessários a seu projeto iOS; criar o objeto de serviço a ser utilizado para fazer solicitações; chamar a API exposta pelo Google Cloud Endpoints; opcionalmente, adicionar suporte para chamadas autenticadas.

Cloud DNS

Servidor DNS confiável, resistente e de baixa latência da rede mundial de servidores Anycast DNS do Google. Criar registros de DNS com uma interface de linha de comando fácil de usar ou programar contra uma RESTful API com funcionalidade total para customizar o serviço para suas necessidades específicas.

Fazer o teste agora

Recursos

Serviço DNS confiável de alto desempenho

Use a infraestrutura do Google para servidores DNS de alto volume para produção de qualidade. Seus usuários terão acesso confiável e de baixa latência à infraestrutura do Google em qualquer lugar do mundo usando nossa rede de servidores de nome Anycast.

Fácil de usar, personalizável às suas necessidades

Use nossa interface de linha de comando programável e fácil de usar para gerenciar seus registros. Ou acesse a RESTful API com funcionalidade total para criar sua interface DNS personalizada.

Gerenciar registros para todos os seus serviços

Conecte os registros DNS para serviços da plataforma Google, como Compute Engine, App Engine ou Google Cloud Storage.

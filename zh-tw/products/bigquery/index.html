BigQuery

BigQuery 可讓您分析儲存在雲端上的巨量資料。您可以執行類似 SQL 的查詢功能，在幾秒鐘之內，便能從數 TB 大小的資料集中找出所需資料。BigQuery 可視需要擴充且簡單易用，可在短時間內提供您資料的分析數據。

立即試用

功能

自動完成一切事務

系統會在背景中以非同步的方式執行您的查詢，而我們可以調查它們的狀態。您可以從 Google Cloud Console 調閱工作紀錄、查詢紀錄以及 Cloud Platform 的其他資源。

輕鬆匯入資料

您可利用 Google Cloud Storage 載入大量資料，或是快速傳輸大量資料 (每秒最多可傳輸 1,000 列資料)。您可以即時處理大量紀錄或交易紀錄，然後快速分析資料。

合理的巨量資料處理費用

Google 提及「巨量資料」時並非誇大其辭。您每個月可免費處理 100 GB 的龐大資料。就算您的資料超出 100 GB 也沒關係，我們可讓您儲存數 TB 的巨量資料，並確保您隨時可加以取用。Google 的定價方式簡單透明，不會隨意向您收取額外費用，費用高低全由您自己決定。Google 只會在您使用我們的儲存空間，或者執行查詢的情況下，才會向您收取費用。

符合所需的介面

我們特別區隔管理介面與開發介面，方便您取用真正需要的工具。您可以透過 Google API Console 控管專案和資料集的存取權。不論您的工作小組有多少人，BigQuery 都可視需要協助完成工作。

價格

BigQuery 使用欄位式資料結構，因此對於每一筆查詢，您只須為各資料欄內處理過的資料付費。而每月處理的前 100GB 資料則完全免費。

價格自 2014 年 4 月 1 日起生效。

如果要處理較大且持續的工作負載，建議您採用預訂容量計費，與以資料量計費相較之下，這種計費方式可節省多達 70％ 的費用。如要申請使用，請與業務代表聯絡。

參考資源

  • 什麼是 BigQuery？

    如果沒有合適的硬體或基礎架構，嘗試查詢大量資料集，須耗費大量的時間和金錢。Google BigQuery 會利用 Google 基礎架構的高處理效率，針對僅供附加資料的表格執行類似 SQL 的查詢，來大幅節省時間和金錢成本。您只要將資料移至 BigQuery 即可，其他繁雜工作就交給 Google 負責。

  • 將資料載入 BigQuery

    開始查詢資料之前，請先將所需資料載入 BigQuery。您可以利用工作或串流紀錄，來載入大量資料。

  • 查詢資料

    查詢是以 BigQuery 的 SQL 語法撰寫而成。BigQuery 同時支援同步和非同步查詢方法，這兩種方法都是由工作來處理，但「同步」方法顯示的逾時值會等到工作完成後才傳回。

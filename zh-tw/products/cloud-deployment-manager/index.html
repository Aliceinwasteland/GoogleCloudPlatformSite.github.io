Cloud Deployment Manager

Cloud Deployment Manager 讓開發人員可利用簡單的陳述式範本，輕鬆設計、共用、部署及管理複雜的 Cloud Platform 解決方案。從簡單的網路伺服器到複雜的高可用性叢集，Deployment Manager 可讓小組成員在管理上花費較少時間，投入更多心力在建設性的工作。

立即試用

功能

在雲端進行簡單、可重複的部署作業

使用 Deployment Manager 的陳述式、可參數化的範本，簡化並重複 Cloud Platform 上的作業，讓營運小組成員不必費力管理其基礎設施的狀態，也無須管理變更內容。

讓 Google 負責維護您的基礎設施

Deployment Manager 可在範本部署期間及其後，主動監控您的資源狀態。它可以監控多種類型的 Cloud Platform 資源，而不必連接眾多 API。

簡單的工具與深度整合

使用我們簡單、可編寫指令碼的指令列介面管理部署作業，或者利用全功能的 OAuth 2.0 相容 RESTful API，深度整合至您專屬的工具鏈和工作流程。

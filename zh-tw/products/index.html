利用 Google 提供的技術和工具執行您的應用程式。
Cloud Platform 提供各種模組，可協助您利用所需服務輕鬆開發應用程式。

運算

Compute Engine

Compute Engine 是 Google 的一項 IaaS (Infrastructure-as-a-Service，基礎架構即服務) 服務。Google 基礎架構所代管的虛擬機器可承接各種大型工作。選擇符合工作需求的虛擬機器，然後讓 Google 遍佈全球的光纖網路為您提供一流服務，永不負使命。我們是以分鐘數計費，絕不會超收費用。

App Engine

App Engine 是 Google 的一項 PaaS (Platform-as-a-Service，平台即服務) 服務。您可利用內建的服務開發應用程式，提高您的工作效率。將您的應用程式部署到由 Google 全盤管理的平台上，其他事就統統交給 Google 完成。您只須下載我們免費提供的 SDK，即可著手開發應用程式，而不必額外支付任何費用。

儲存空間

Cloud SQL

利用完全託管的關聯式 MySQL 資料庫，儲存及管理資料。Google 負責進行複製作業、更新託管項目以及管理資料庫，確保資料庫可正常提供服務。

  • 功能
  • 個案研究
  • 價格

Cloud Storage

利用穩定耐用的物件儲存服務。使用者可利用全球 Edge Caching 技術，在世界任何角落存取您應用程式的資料。Google 負責管理各種版本、確保 SLA 的可靠性，另外還提供 API，讓您可透過程式設計的方式管理自己的資料。

Cloud Datastore

Cloud Datastore 提供一種託管的 NoSQL 無結構資料庫，可用於儲存不具關聯性的資料。Cloud Datastore 會在您需要時，自動擴大儲存空間，此外還支援交易事務以及類似 SQL 的查詢功能。

  • 價格

巨量資料

BigQuery

BigQuery 可讓您分析儲存在雲端上的巨量資料。您可以執行類似 SQL 的查詢功能，在幾秒鐘之內，便能從數 TB 大小的資料集中找出所需資料。BigQuery 可視需要擴充且簡單易用，可在短時間內提供您資料的分析數據。

服務

Cloud DNS

DNS 服務是由 Google 的 Anycast DNS 伺服器全球網路所提供，具備穩定、有彈性且低延遲等優點。您可以透過簡單易用的指令列介面建立 DNS 紀錄，也可以針對全功能的 RESTful API 進行程式設計，根據特定需求自訂服務。

  • 功能
  • 價格
  • 說明文件

Cloud Endpoints

自行撰寫 RESTful 服務，並供使用者透過 iOS、Android 和 JavaScript 用戶端加以使用。自動產生用戶端程式庫，輕鬆建置前端平台。內建的重要基礎架構包括拒絕服務 (denial-of-service) 攻擊防護、OAuth 2.0 支援以及用戶端金鑰管理。

Translate API

建立多語言應用程式，利用電腦翻譯功能，將文字翻譯成其他語言。目前支援的語言高達數千種。

Prediction API

利用 Google 的機器學習演算法，並搭配熟悉的 RESTful 介面，即可分析及預測結果。

管理

Cloud Deployment Manager

Cloud Deployment Manager 讓開發人員可利用簡單的陳述式範本，輕鬆設計、共用、部署及管理複雜的 Cloud Platform 解決方案。從簡單的網路伺服器到複雜的高可用性叢集，Deployment Manager 可讓小組成員在管理上花費較少時間，投入更多心力在建設性的工作。

開發人員工具

Google Cloud SDK

Google Cloud SDK 提供工具與資料庫，可讓您在 Google Cloud Platform (包括 App Engine、Compute Engine、Cloud Storage、Cloud SQL 和 BigQuery) 上建立及管理資源。

推送部署

利用 Git 自動將應用程式部署到 App Engine。您只要在存放區推送新的版本，即可加以發佈，然後就可以使用 Git 的所有功能，包括提交紀錄。

Cloud Playground

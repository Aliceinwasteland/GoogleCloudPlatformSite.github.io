Google Cloud Storage - Cloud Backup, Cloud Database & Online Storage — Google Cloud Platform

Cloud Storage

利用穩定耐用的物件儲存服務。使用者可利用全球 Edge Caching 技術，在世界任何角落存取您應用程式的資料。

立即試用

功能

安全無虞

我們會將您的資料保存在多個位置的儲存空間，並採用 OAuth 和存取權控管制度，確保您資料的安全。

超越市場、具彈性的收費方式

選擇符合自己資料需求的儲存空間解決方案，包括標準儲存空間或 DRA 版雲端硬碟。與其他 Cloud Platform 一樣，收費方式為用多少付多少。

具備各種 API 的物件儲存空間

雖然利用簡單的 API 就可以使用 Cloud Storage，但是各種先進的功能，可提高彈性和處理能力。物件版本、平行上傳以及 CRC 式完整性檢查功能，可協助開發人員建立穩定、實用的應用程式。您可直接利用 XML 或 JSON，或是利用數個主流程式設計語言的程式庫，來使用這個 API。

靈活運用

您可以直接從 Cloud Storage 提供靜態物件，或是將 Cloud Storage 做為其他 Google 產品的資料存放區。 BigQuery、App Engine 或其他 Cloud Platform 服務使用您的儲存空間。您可利用盛名全球的 RESTful API 取得您的資料，這種 API 速度快、設計靈活，還具備可確保安全性的 OAuth 技術。

立即使用

請參閱專案範例以及採用 Google Cloud Storage 的企業所提供的使用心得。

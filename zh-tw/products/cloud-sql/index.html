Cloud SQL

利用完全託管的關聯式 MySQL 資料庫，儲存及管理資料。Google 負責進行複製作業、更新託管項目以及管理資料庫，確保資料庫可正常提供服務。

立即試用

功能

在雲端平台部署 MySQL 資料庫十分輕鬆簡單。Google Cloud Platform 提供功能強大的資料庫，執行速度快且具備大量儲存空間，可為您的應用程式提供額外的可靠儲存空間。

熟悉的基本架構

Cloud SQL 提供的標準 MySQL 資料庫，是廣受大眾歡迎的開放原始碼資料庫，可協助您在雲端平台上快速建置及部署應用程式。MySQL 資料庫執行個體最多可配備 16 GB 的 RAM，以及 100 GB 的儲存空間。

具彈性的收費方式

我們所採用的「用多少付多少」收費方式，可讓您以低廉的價格，享受高品質的服務。如果您不常使用資料庫，Google 會按照實際使用時間向您收取費用，避免您花冤枉錢。您可以針對工作量繁重的執行個體購買支援方案，來控制成本。

安全、穩定又耐用

我們會將您的資料複製到多個位置，當某處資料庫發生故障時，另一處的資料庫就會自動接手處理，為您提供服務。這表示您的資料絕對安全無虞，且即使發生嚴重的軟硬體故障狀況，也能從資料庫存取您的資料。此外，Google 也會管理您的備份，讓您需要還原以前的資料時，馬上就可以取得備份。

恣意遷移資料

您可利用標準連線和 mysqldump、MySQL Wire Protocol 和 JDBC 等此類工具，來遷移資料。Google Cloud Platform 可協助您將資料遷移至任何位置。

控制項

透過雲端主控台或 JSON API 輕鬆管理及使用您的執行個體。

全方位管理

複製資料、更新管理化項目或管理資料庫等繁雜工作，全交給 Google 負責。

價格

Google 為 Cloud SQL 提供兩種計費方式：

  • 針對流量偏高的開發人員，Google 推出價格低廉且可事先規劃預算的方案方案。
  • 針對流量較低的開發人員，Google 推「用多少付多少」的彈性收費方式。也就是說，Google 會根據您的實際使用時間收取費用。建議您使用雲端平台所代管的 MySQL 資料庫，每月最低只要 $1。

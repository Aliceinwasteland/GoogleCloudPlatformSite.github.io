Benefits of Using Google Cloud Platform — Google Cloud Platform

選用 Google Cloud Platform 的原因

開發人員可利用 Google Cloud Platform，藉由我們彈性十足、穩定可靠的基礎架構，建置、測試及部署應用程式。Google Cloud Platform 提供適用於網路、行動和後端平台解決方案的運算、儲存以及應用程式服務。

採用 Google 的基礎架構

如果您將相同基礎架構做為建置環境，Google 便能夠在極短的時間內傳回數十億筆搜尋結果，每月提供 600 億小時的 YouTube 影片時間，以及為 4 億 2,500 萬位 Gmail 使用者供應儲存空間。

全球網路：Google 擁有全球最大以及最先進的電腦網路。Google 的中樞網路擁有數千英里的光纖纜線、使用最先進的軟體定義網路技術，並且具備 Edge Caching 服務，可提供快速、穩定且可視需要擴充的處理能力。事實上，我們還在太平洋海底鋪設了光纖纜線。

完善的資料備份支援：我們在全球各地設立多個資料中心，以提供完善的資料備份支援。您的資料會自動備份到全球各地的儲存裝置。

最先進的電腦科技：基礎架構的創新並不是指硬體而已。Google 在軟體基礎架構方面引領業界，例如 MapReduce、BigTable 和 Dremel。 而現在，Google 將隆重推出下一代的創新軟體，例如 Spanner 和 Flume。如果您有在 Google Cloud Platform 上建置應用程式，即可搶先試用 Google 研發出的創新科技。

「Google 能夠建置、管理大型伺服器網路和光纖纜線，並讓它們保持正常運作，其作業速度和效率是同業其他公司無法比擬的...。Google 之所以能夠在現今社會中大放異彩，是因為它成功建置了實體網路、數千英里的光纖纜線，以及數千台伺服器，而這些都是成就雲端平台的幕後功臣。」

  • - Wired Magazine
  • 資料中心虛擬導覽

全心投入產品的開發作業

快速開發及部署應用程式，並在需要時重複這兩個步驟，而不必費神從事系統的管理作業。Google 會管理您的應用程式、資料庫和儲存空間伺服器，為您省下大量時間。

管理化服務：Google 負責資料庫管理、伺服器設定、資料分割以及負載平衡等繁雜工作，讓您得以專心撰寫程式碼。從此您再也不必隨時待命，也不必撰寫重複不變的程式碼。

開發人員工具和 SDK：Google 整合各種熟悉的開發工具 (例如 Eclipse)，並且提供 API 用戶端程式庫以及命令行介面，以協助您輕鬆設計出各種應用程式。

主控台和系統管理：Google 提供了一個控制台，可讓您查看及管理所有的應用程式。您可以使用單一簡易介面，查看應用程式的效率狀況，同時管理帳戶和帳單資訊。

混合及配對服務

舉凡虛擬機器、管理化平台、Blob 儲存空間、資料庫儲存空間、NoSQL 資料庫、MySQL 資料庫、巨量資料分析等等應用程式架構所需的服務，Google Cloud Platform 皆可提供。

運算：Google Cloud Platform 提供由 Google 全盤管理的平台，以及具彈性的虛擬機器，協助您選擇符合自身需求的系統。如果您只想全心投入程式碼的設計作業，而不想從事系統的修正和維護工作，請使用我們的 PaaS (Platform-as-a-Service，平台即服務) 服務 App Engine。您可使用 Compute Engine 提供的原始虛擬機器，設計出所需的任何實體。

儲存空間：Google Cloud Platform 提供各種儲存服務，讓您輕鬆管理及存取資料。您可以透過 Cloud SQL 和 Datastore，來使用 MySQL 或無結構 (Schemaless) NoSQL 資料庫，此外，Cloud Storage 提供全球 Edge Caching 功能，可讓您自由選擇物件儲存空間。

服務：利用 Google API 和服務，為您的應用程式快速開發各種功能。您不需要從頭建置這些服務，只要利用 Google Cloud Platform 便利的整合功能即可。

靈活調整處理能力，因應服務數百萬使用者的流量需求

當網際網路流量變得十分龐大時，Google Cloud Platform 所代管的應用程式會自動擴大處理能力，而在流量下降之後，便會調節處理能力，達到用多少，付多少的目的。

擴大支援能力：Cloud Platform 能夠擴大產品的支援能力，例如 Google 的產品，即便使用人數相當龐大，也可以從容應付。像 App Engine 或 Cloud Datastore 此類的管理化服務，都可以自動擴大支援能力，以便在當使用者人數越來越多時，讓您的應用程式得以承受流量。

調降支援能力：如同 Cloud Platform 可讓您擴大支援能力一樣，管理化服務同樣可以調降支援能力。您不必為用不著的運算資源，支付任何費用。

效能值得信賴

Google 不容許絲毫的延遲，為此我們的計算基礎架構採用最好的 CPU、記憶體和磁碟。我們的網路和 Edge Cache 則可以快速回應全球各地的使用者。

CPU、記憶體、磁碟：Google Cloud Platform 在運算、儲存以及應用程式服務方面，提供快速、一致的效率。有了超強的處理能力、所需的記憶體以及高速 IOPS，您的應用程式就可以為使用者提供一致的效率。您不必擔心出現延遲狀況，或是發生「noisy-neighbor」問題。

全球網路：Google 使用軟體制式網路技術，封包可以繞經全球以及提供快速的 Edge Caching 功能，確保您的資料可傳送給需要的使用者。儘管延遲狀況是以毫秒為單位計算，不過 Google 可保證您的內容一定能夠即時傳送給使用者。

無須維護：利用新的內建動態遷移技術，虛擬機器再也不需要在進行定期維護作業時暫停服務。Google 會自動安裝主機的修正檔案，而且維護資料中心時不會暫停服務，請您放心。

Sebastian Stadil

CEO Scalr

「Google Cloud Platform 的效能不會時好時壞，這是有目共睹的。每個 VM 和磁碟的效率，完全符合我們期望的水準，讓我們得以建置快速、低延遲的應用程式。」

隨時提供您需要的支援

Google 涵蓋全球各地的使用者社群、合作廠商供應鏈以及高級支援套件，可提供您所需的各種資源。

快速上手：Google 備有各種資源，讓您搶先體驗 Google Cloud Platform 的高效率。技術文件、架構範例以及解決方案可為您提供一流應用程式的設計模組。

加入社群：請加入開發人員社群，瞭解他們如何使用 Google Cloud Platform 建置一流的應用程式。您也可以前往 Stack Overflow 網站尋求支援，或是加入我們的 Google+ 社群，直接與其他 Google Cloud Platform 使用者交流互動。

高級支援：Google Cloud Platform 提供一系列全年無休的支援方案，請根據自己的需求加以選購。選購前請詳閱各種支援套件的說明。

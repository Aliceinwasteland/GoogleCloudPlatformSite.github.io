300 秒部署一款應用程式

瞭解如何利用 Google Cloud Endpoints 撰寫程式碼，以快速開發行動後端平台。系統為 iOS、Android 和 JavaScript 自動產生的用戶端程式庫，可協助呼叫後端平台 API 。

在 Google Cloud Platform 上建置行動應用程式有何優點：

自動擴充及負載平衡功能

您唯一要做的事就是撰寫程式碼，其他事交給 Google 負責。無論使用者人數多麼龐大，App Engine 皆可自行進行擴充。

提供靜態內容

App Engine 從應用程式伺服器之外的專用伺服器與快取提供靜態檔案。

資料處理

您可以透過應用程式紀錄，瞭解應用程式的使用人數，以及後端平台的行為。BigQuery 可讓您即時對紀錄檔案進行隨機分析，而且分析數 TB 的資料只需幾秒鐘的時間。

搜尋附近地區的資料

利用 App Engine 的 Search API，搜尋附近地區的資料。

推播通知

利用 Google Cloud Messaging，將資料傳送給 Android 使用者。或者利用 Sockets API，將推播通知傳送給 iOS 使用者。

資料儲存空間

您可以利用簡易的 REST 介面，將物件和檔案儲存到 Google Cloud Storage，即使大小高達數個 TB，一樣沒問題。

如何開始

如要部署行動應用程式，您只要取得程式碼，然後參閱教學課程以及 Google Cloud Platform 的指示即可。

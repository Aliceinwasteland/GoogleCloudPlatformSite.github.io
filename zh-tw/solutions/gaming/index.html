解決方案

提供大型媒體檔案

越來越多使用者重視應用程式可否提供高解析度和各種大小的圖片。為此，您必須持續更新應用程式的內容，以吸引使用者再次使用。Google Cloud Storage 除了提供簡易的 REST 介面，可讓您儲存大型檔案之外，更採用與 Google 的應用程式相同的高速網路，讓您能夠快速取得所需資料。

分析使用者參與度

想透過應用程式獲得利潤，就必須瞭解使用者的遊玩行為，以及您遊戲的受歡迎程度。不過，要想達到上述目標，除了須記錄相關資料，您還需要可靠的數據分析功能。Google BigQuery 提供 Google 各項服務所採用的技術，可協助您解決應用程式的問題，以便針對數 TB 的資料即時進行隨機分析，來取得各種數據。或者，您可以在 App Engine 上使用 MapReduce，平行處理大量資料集，來進行非即時分析。

同步處理多個裝置的遊戲狀態

將遊戲狀態儲存至 App Engine Datastore 或 Google Cloud Storage，方便使用者透過各種裝置加以存取，並且讓您得以仔細分析遊戲狀態。

輕鬆與行動裝置進行整合

您可利用 Google Cloud Endpoints，將伺服器邏輯自動轉換為網路 API，然後為產生 iOS、Android 和 JavaScript 版本的用戶端程式庫，以便您開發標準網路應用程式。

在 Google Cloud Platform 上建置各種遊戲

您可以在 Google Cloud Platform 上建置適用於行動平台 (例如 Android 和 iOS)，以及適用於網路平台 (例如 Chrome OS 和 Facebook) 的遊戲。您也可以建置不同類型的遊戲，例如回合制遊戲或社交遊戲。

利用 Google 銷售遊戲及賺取利潤

利用 Google AdSense 和 Google 應用程式內建付款功能賺取利潤。您可以透過 AdSense 放送與遊戲主題有關的廣告。當您透過應用程式內建付款功能銷售虛擬商品時，須支付 5% 的手續費。您可透過 Chrome Web Store 或是 Google Play，來銷售網路遊戲或行動遊戲。

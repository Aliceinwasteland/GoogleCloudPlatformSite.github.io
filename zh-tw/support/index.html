支援

支援方案

銅級支援

我們為所有的用戶提供銅級支援，您可以查閱線上說明文件、進入社群論壇以及享有計費功能支援服務。

銀級支援

可讓您直接向支援小組洽詢服務功能、實用架構以及服務錯誤等相關問題。

金級支援

可讓您享有全年無休的電話客服支援、更快速的首次回覆時間及應用程式開發諮詢，以及符合您需求的架構支援。

白金級支援

可讓您享有 Google 為您量身打造的全方位專業技術支援，其中包括金級支援的各項服務，以及直接洽詢技術支援客戶經理小組的權限。

申請銀級或金級支援

想申請白銀級支援嗎？與銷售人員聯絡

請按照以下步驟快速申辦銀級或金級技術支援方案。


  1. 登入 Cloud Console。
  2. 在左側選單中，選取 [專案]。
  3. 在 [專案名稱] 下方點選專案名稱。
  4. 在左側選單中，選取 [支援]。
  5. 系統隨即顯示您目前使用的支援方案。
  6. 選取 [選擇銀級] 或 [選擇金級]。
  7. 選取 [啟用計費功能]，然後按照系統提示的步驟進行。


*如果系統未顯示變更連結，請先將應用程式與 Cloud Console 整合。方法是登入 App Engine Console，點選左側選單中「管理」部分下方的 [應用程式設定]，然後點選網頁底端的 [新增專案]。

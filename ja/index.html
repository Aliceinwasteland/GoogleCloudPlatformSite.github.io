最新アプリケーションのためのツール

Google Cloud Platform を利用することにより、ユーザはスケーラビリティと信頼性の富んだGoogleのインフラ上でアプリケーションの作成、テスト、さらに、デプロイすることができるようになります。ウェブ、モバイル、バックエンドの用途別に、それぞれに最適なコンピューティング、ストレージ、アプリケーションの各サービスを提供しています。

今すぐ試す

Google Compute Engine の正式版がリリースされました

Google Compute Engine が、月間 SLA 99.95%、年中無休のサポートで正式版としてご利用いただけるようになりました。ライブ マイグレーションによってメンテナンス期間をなくし、価格を 10% 引き下げ、Red Hat、SUSE、FreeBSD などの Linux バリアントのサポートを追加し、16 コアの新しいインスタンスを導入しました。

Google Cloud Platform は 400 万以上のアプリケーションで使用され信頼を得ています

Google を選ぶ理由

Google が利用しているインフラと同一のインフラをユーザ向けに提供

Google Cloud Platformのユーザは、瞬時に数十億件の検索結果を返し、月に60 億時間YouTube 動画を再生し、4 億 2,500 万人に及ぶ Gmail ユーザーにストレージを提供しているGoogleのインフラと同じインフラを利用することができるようになるのです。Googleが提供するインフラの主な優れた点は以下の通りです。

詳細

開発のみに集中可能

システム管理を気にすることなく、アプリケーションをすばやく開発、デプロイ、反復できます。Google が利用者に代わってアプリケーション、データベース、ストレージ サーバーを管理します。

詳細

サービスの選択と最適化

仮想マシン、マネージドプラットフォーム、BLOB ストレージ、ブロック ストレージ、NoSQL データストア、MySQL データベース、ビッグ データ分析。Google Cloud Platform には、アプリケーション アーキテクチャが必要とするサービスがすべて揃っています。

詳細

ユーザー数に応じてスケールアップ

Cloud Platform でホストされているアプリケーションは、大量のインターネット規模のワークロードを処理する際には自動的にスケールアップされ、トラフィックが低下するとスケールダウンされます。料金は、使用したものだけに支払います。

詳細

信頼性に優れたパフォーマンス

遅延はミリ秒単位でも重大な問題です。Google のコンピューティング インフラストラクチャは、CPU、メモリ、ディスクの全てにおいて、安定したパフォーマンスを提供します。Google のネットワークとエッジ キャッシングは、世界中のユーザーに迅速なレスポンスを提供します。

詳細

使用開始に当たり、多方面から支援します

世界中のユーザー コミュニティ、パートナー エコシステム、プレミアム サポート パッケージなど、Google は皆様が Google Cloud Platform 使用を開始し、実際にビジネスにご活用頂けるよう、多方面からご支援させていただきます。

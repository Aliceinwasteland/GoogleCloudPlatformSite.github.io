Cloud Datastore — Google Cloud Platform

Cloud Datastore

スキーマレスな NoSQL 管理型データベースを使用して、非リレーショナル データを保存できます。Cloud Datastore は必要に応じて自動的に拡張し、トランザクションだけでなく堅牢な SQL に似たクエリにも対応しています。

今すぐ試す

特長

SQL に似たクエリによるスキーマレスなアクセス

データ モデルと移行を心配する必要はありません。Cloud Datastore はスキーマレスなストレージ サービスであり、データの基盤構造を考慮する必要がないため、機敏な対応が可能となります。Cloud Datastore は、複数のプロパティにまたがってデータを検索し、必要に応じて並べ替えることができる堅牢なクエリ エンジンを提供します。

管理型データベース

Cloud Datastore は完全管理型のデータベースです。高可用性と一貫性のあるデータベースを提供するために、シャーディングとレプリケーションは Google で自動的に処理されます。

ユーザーに応じた自動拡張

Cloud Datastore はニーズに応じて自動的に拡張します。そのため、プロビジョニングや負荷の予測を心配することなく、アプリケーションのビルドに集中できます。

ACID トランザクション

Cloud Datastore は、楽観的並行性制御を使用して ACID トランザクションを提供します。アプリケーションは、複数のデータストア操作を 1 つのトランザクションで実行できます。結果はすべて成功するか、すべて失敗するかのいずれかになり、データの整合性が保証されます。

冗長性をビルトイン

1 回の API コールで、複数のデータ センターをまたがってデータの複製が自動的に作成されます。高い可用性と耐久性が基盤全体にビルトインされています。

ローカルでの開発用のツール

Cloud Datastore 開発キットを使用すると、アプリケーションを完全に展開することなくローカルで開発、テスト、反復処理できます。

どこからでもデータにアクセス

統合ポイントを Cloud Datastore に依存し、App Engine と Compute Engine にまたがったソリューションをビルドできます。Cloud Datastore で公開される RESTful インターフェース を使用することにより、どの展開ターゲットからでもデータに簡単にアクセスできます。

リソース

  • Google Cloud Datastore の概要

    Google Cloud Datastore は、Google Cloud Compute Engine インスタンスまたは他の場所で実行中のクライアントから Google Cloud Datastore にアクセスして管理するための一連のツールとクライアント ライブラリを提供します。Google Cloud Datastore が提供する RPC ベースの API は、Google マップ、YouTube、Google Apps API をホストするのと同じ Google サービスによってホストされています。この API は、ほとんどのアプリや言語に対応しており、たとえば仮想マシンやオンプレミス Java アプリケーション サーバーで実行される node.js ウェブ アプリと共に使用できます。

  • Google Cloud Datastore のアクティベーション

    Google Cloud Datastore API をアクティベーションする方法、この API に関連付けられている割り当て、課金を有効にする方法について説明しています。アクティベーション処理は、既存の設定および環境によって変わる可能性があります。

  • データストアの主要な概念

    Google Cloud Datastore は、アプリケーションを保管するための堅牢かつ拡張可能なストレージを提供するスキーマレスなオブジェクト データストアです。計画休止時間の排除、アトミック トランザクション、読み書き操作の高可用性、読み取りと上位クエリに対する強整合性、その他すべてのクエリに対する結果整合性といった機能を備えています。

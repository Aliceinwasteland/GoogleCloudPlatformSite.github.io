Cloud Deployment Manager

Cloud Deployment Manager では、シンプルな宣言型テンプレートを使って、複雑な Cloud Platform ソリューションを簡単に設計、共有、デプロイ、管理できます。単純なウェブ サーバーでも複雑な高可用性クラスタでも、Deployment Manager を利用すれば、管理にかかる時間を減らして構築に多くの時間をかけられるようになります。

特長

Cloud Platform でデプロイを簡単に繰り返し可能

Deployment Manager のパラメータ化可能な宣言型テンプレートにより、Cloud Platform での操作が簡単かつ繰り返し可能となり、インフラストラクチャの状態管理や変更の管理の負担から解放されます。

インフラストラクチャの監視を Google に任せる

Deployment Manager では、テンプレートのデプロイ中とデプロイ後の両方において、リソースの状態を積極的に監視できます。さまざまな Cloud Platform リソースのタイプに対応可能なため、多数の API を組み合わせる必要がなくなります。

簡単なツールと密接な統合

シンプルでスクリプト可能なコマンドライン インターフェースを使ってデプロイを管理でき、また、フル機能の OAuth 2.0 対応 RESTful API を利用して、ツールチェーンやワークフローへの密接な統合が可能になります。

Translate API — Google Cloud Platform

Translate API

Google 翻訳と統合することで、対応する数千の言語ペア間をすばやく動的に翻訳できます。

今すぐ試す

特長

言語に動的にアクセス

Google Translate API は、多数の言語をサポートしています。RESTful API を使用して、ドキュメントの言語を検出し、翻訳を行います。

Google API を使用してアクセス可能

Translate API には標準の Google API を使用してアクセスできます。Python、Objective C、Ruby など 10 個のプログラミング言語のサンプル コードとライブラリをご覧ください。

手頃な料金設定

Translate API の料金は、送信された文字数（1,000,000 文字）に基づいています。料金体系はわかりやすく、手頃な設定になっています。

料金

Google Translate API の料金は使用量に基づいています。翻訳の使用量は 1,000,000 文字単位（M）で計算されます。1 M = 106 文字となります。詳しくは、料金に関するよくある質問をご覧ください。

重要: Google Translate API v2 では、このサービスの使用を開始する前にすべてのアカウントのお支払い情報が必要となります。課金を有効にする手順については、下記をご覧ください。

  • 使用料:
    • 翻訳:
      • テキスト 1,000,000 文字につき 20 ドル。課金は実際に提供された文字数に比例して調整されます。たとえば、翻訳する文字数が 500,000 文字だった場合は、10 ドルが課金されます。
    • 言語の検出:
      • テキスト 1,000,000 文字につき 20 ドル。課金は実際に提供された文字数に比例して調整されます。
  • 使用制限:
    • Google Translate API はデフォルトで 1 日 2,000,000 文字に制限されています。この上限は、Cloud Console で下記の手順を行うことで 1 日 50,000,000 文字までに増やすことができます。
    • 1 日 50,000,000 文字以上翻訳する必要がある場合は、お問い合わせください。

プロジェクトの課金を有効にするには、以下の手順を行います:

  1. Google Cloud Console に移動します。
  2. プロジェクトを選択します。
  3. 左にあるサイドバーで [課金] を選択します。
  4. ページの上部で [課金を有効にする] を選択します。
  5. フォームに入力します。

プロジェクトの使用量上限を確認、変更する場合、または上限合計を増やすよう Google にリクエストする場合は、以下の手順を行います:

  1. Google Cloud Console に移動します。
  2. プロジェクトを選択します。
  3. 左にあるサイドバーで [API と認証] を選択し、API を選択します。
  4. API の情報ページで、API 名の近くにある [割り当て] を選択します。

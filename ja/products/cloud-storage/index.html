Cloud Storage

高い可用性と永続性を備えるオブジェクトストレージ サービスです。グローバルなエッジキャッシングにより、ユーザーはどこからでもアプリのデータにすばやくアクセスできます。

今すぐ試す

特長

安心のセキュリティ

複数の地域に配置されている冗長ストレージによりデータを保護します。きめ細かなアクセス制御と OAuth により、カスタマイズ可能で強力なセキュリティを実現できます。

柔軟でリーズナブルな料金体系

標準ストレージまたは DRA（中可用性永続）ストレージのうち、データの特性に合ったストレージ ソリューションを選べます。他の Cloud Platform サービスと同様、支払いは従量制です。

オブジェクト ストレージと多機能 API

Cloud Storage は簡単にアクセスできるシンプルな API に加えて、高い柔軟性とパフォーマンスを備えた高度な機能セットを提供します。オブジェクトのバージョン管理、並列アップロード、CRC ベースの整合性チェックにより、堅牢で高度なアプリケーションの開発が可能です。XML や JSON を使用して API に直接アクセスできるほか、一般的なプログラミング言語のライブラリも利用できます。

柔軟なアクセス

Cloud Storage は静的オブジェクトの直接配信に利用できるほか、他の Google サービスのデータストアとしても使用できます。BigQuery、App Engine などの Cloud Platform サービスからアクセスできます。スピードと柔軟性、および OAuth による認証機能を備えた RESTful API を使用してデータにアクセスできます。

  • Google Cloud Storage の概要

    Google Cloud Storage には、世界中のどこからでも、いつでもデータを保存、取得できます。データの量に制限はありません。シンプルなプログラミング インターフェースを備えており、高速で信頼性の高い Google のネットワーキング インフラストラクチャを利用して開発できるため、安全で費用対効果の高いデータ オペレーションを実現できます。容量拡大の必要が生じた場合も、Google のインフラストラクチャに備わる高い拡張性のメリットが得られます。

Cloud SQL

データの保存や管理に利用できる、フルマネージドの MySQL データベースです。レプリケーション、パッチ管理、データベース管理は Google が行うため、高い可用性とパフォーマンスが得られます。

今すぐ試す

特長

MySQL データベースをクラウド上で簡単に利用できます。高速かつ強力なデータベース機能を提供し、容量が不足することもありません。アプリケーションに必要な冗長性と信頼性を備えるストレージとしてご利用いただけます。

使いやすいインフラストラクチャ

Cloud SQL は、世界中で幅広く利用されているオープンソース データベースである標準の MySQL データベースを備えているため、クラウドでのすばやい開発が可能です。インスタンスでは、最大 16 GB の RAM と 100 GB のデータ ストレージを使用できます。

柔軟な料金体系

従量制であるため、手軽に使い始められます。実際にデータアクセスした時間に対してのみ請求が発生するため、データベースの使用頻度が高くない場合のコストを抑えられます。より高負荷のインスタンスのコストを節約できるパッケージ オプションも利用できます。

安全性、可用性、耐久性

データはデフォルトで複数の地域で複製され、フェイルオーバーも自動的に行われます。このため、データは厳重に保管され、重大な障害が発生した場合でもデータベースにアクセスできます。バックアップの管理も Google が行うため、必要に応じて簡単に復元できます。

ロックインのない簡単なマイグレーション

mysqldump、MySQL ワイヤ プロトコル、JDBC などの標準的な接続やツールをサポートし、Google Cloud Platform と他プラットフォーム間の移行を簡単かつ柔軟に行えます。

制御

Cloud Console や JSON API を使用して、インスタンスの管理やインスタンスへのアクセスを簡単に行えます。

完全管理

複製、パッチ管理、データベース管理などの作業を Googleにお任せください！

料金

Cloud SQL では次の 2 つのお支払いプランを利用できます:

  • トラフィック量の多いアプリケーション向けには、ディスカウント付きのパッケージ プランを用意しています。コストの事前見積もりが容易です。
  • 軽量なアプリケーション向けには、より柔軟な従量制の料金体系を用意しています。データにアクセスした場合にのみ料金が発生し、クラウド上の MySQL データベースを月額約 $1 から使い始めることができます。

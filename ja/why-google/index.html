Google Cloud Platform を選ぶ理由

Google Cloud Platform を利用することにより、ユーザはスケーラビリティと信頼性の富んだGoogleのインフラ上でアプリケーションの作成、テスト、さらに、デプロイすることができるようになります。ウェブ、モバイル、バックエンドの用途別に、それぞれに最適なコンピューティング、ストレージ、アプリケーションの各サービスを提供しています。

Google が利用しているインフラと同一のインフラをユーザ向けに提供

Google Cloud Platformのユーザは、瞬時に数十億件の検索結果を返し、月に60 億時間YouTube 動画を再生し、4 億 2,500 万人に及ぶ Gmail ユーザーにストレージを提供しているGoogleのインフラと同じインフラを利用することができるようになるのです。
Googleが提供するインフラの主な優れた点は以下の通りです。

グローバル ネットワーク: Google が提供するコンピュータネットワークは世界でも最大かつ最も革新的なものの一つです。数千マイルの光ファイバー ケーブルや先進的な SDN（Software Defined Network）、また、エッジ キャッシング サービスを利用しており、高速かつ安定し、スケーラブルなパフォーマンスを実現しています。Google は太平洋の下に 独自 の光ファイバー ケーブルを敷設しています。

冗長性: 世界中に複数のPOP（Multiple Points of Presence）を配置することににより、強力な冗長性が実現されています。データは複数の場所にあるストレージ に自動的にミラーリングされます。

最新のコンピュータ サイエンス: インフラストラクチャの革新はハードウェアだけに限りません。Google は、MapReduce、BigTable、Dremel などのソフトウェア インフラストラクチャの革新で業界をリードしてきました。現在 Google では、Spanner、Flume などのサービスで次世代の革新を推進しています。Cloud Platform を利用することにより、Google の革新的技術へより速くアクセスできるようになります。

「優れた効率性とスピードで、サーバーの巨大なネットワークと光ファイバー ケーブルを作成、体系化、運用できる Google の機能。これこそが、Google の Google たる所以です。物理ネットワーク、何千マイルものファイバー ケーブル、何千台ものサーバー全体が、すべてのクラウドの基盤となっています。」

開発のみに集中可能

システム管理を気にすることなく、アプリケーションをすばやく開発、デプロイ、反復できます。Google が利用者に代わってアプリケーション、データベース、ストレージ サーバーを管理します。

マネージドサービス： データベース管理、サーバー設定、シャーディング（分散）、ロード バランシングは Google に任せてコーディングに集中できます。ページングの実行やボイラープレート コードの作成は不要です。

デベロッパー ツールと SDK （Software Development Kit）: Google では Eclipse などの使いやすい開発ツールと統合し、API クライアント ライブラリとコマンドライン インターフェースを提供しているため、思いどおりのものが簡単に作成できます。

コンソールと管理: 1 つのコンソールからすべてのアプリケーションを確認して管理できます。シンプルなインターフェースでアプリケーションのパフォーマンスを確認し、アカウントや支払いを管理できます。

サービスの選択と最適化

仮想マシン、マネージドプラットフォーム、BLOB ストレージ、ブロック ストレージ、NoSQL データストア、MySQL データベース、ビッグ データ分析。Google Cloud Platform には、アプリケーション アーキテクチャが必要とするサービスがすべて揃っています。

コンピューティング: Cloud Platform の完全に管理されたプラットフォームとフレキシブルな仮想マシンにより、ニーズに合ったシステムを選択できます。パッチやメンテナンスを気にせずにコードに集中したい場合は、PaaS の App Engine をご利用ください。Compute Engine で RAW 仮想マシンにアクセスすれば、柔軟に必要なものを作成できるようになります。

ストレージ: Google Cloud Platform では、簡単ですばやいデータアクセスを維持できるさまざまなストレージ サービスを提供しています。Cloud SQL とDatastore を使用すると、MySQL データベースやスキーマレスな NoSQL データベースを利用できるようになります。また、Cloud Storage では、グローバルなエッジ キャッシングでフレキシブルなオブジェクト ストレージを提供します。

サービス: Google API とサービスを使用すると、アプリケーションの広範な機能をすばやく有効にできます。最初から作成する必要はなく、Cloud Platform 内に組み込まれている機能を活用するだけです。

使用量に応じてスケールアップ＆スケールダウン

Cloud Platform は、アクセスや処理量の増加に伴うスケールアップ、低下に伴うスケールダウンに対応します。料金は、従量課金性。ご利用いただいた分だけお支払い頂きます。

スケールアップ: Cloud Platform は、トラフィックが急増した場合、それに応じて拡張されるよう設計されています。App Engine や Cloud Datastore などのマネージドサービスは自動スケーリングに対応しているため、ユーザが作成したアプリケーションもそれを使う人の数やアクセス量に応じて拡張されます。

スケールダウン: Cloud Platform でスケールアップができるように、マネージドサービスではスケールダウンも可能になっています。不要なコンピューティング リソースに料金を支払う必要はありません。

信頼性に優れたパフォーマンス

遅延はミリ秒単位でも重大な問題です。Google のコンピューティング インフラストラクチャは、CPU、メモリ、ディスクの全てにおいて、安定したパフォーマンスを提供します。Google のネットワークとエッジ キャッシングは、世界中のユーザーに迅速なレスポンスを提供します。

CPU、メモリ、ディスク: Google Cloud Platform では、コンピューティング、ストレージ、アプリケーションのさまざまなサービスにわたり高速で安定したパフォーマンスを提供しています。パワフルな処理能力、必要なメモリへのアクセス、高い IOPS により、アプリケーションはユーザーに安定したパフォーマンスを提供します。遅延を減らし、他のアプリケーションからの影響も避けることができます。

グローバル ネットワーク: Google では、SDN（Software Defined Network）のテクノロジーを使用して世界中でパケットをルーティングするとともに、高速のエッジ キャッシングを有効にしているため、ユーザーに最も提供しやすい場所にデータが自動的に配置されます。ミリ秒単位の遅延がクリティカルになる環境下、Google が迅速なコンテンツ配信を可能にします。

透過性に優れたメインテナンス: 最新のライブ マイグレーション テクノロジーが組み込まれていることで、計画メンテナンスによって仮想マシンがダウンすることはありません。ホストにはパッチが適用され、データ センターはダウンタイムに悩まされることなく維持されますのでご安心ください。

使用開始に当たり、多方面から支援します

世界中のユーザー コミュニティ、パートナー エコシステム、プレミアム サポート パッケージなど、Google は皆様が Google Cloud Platform 使用を開始し、実際にビジネスにご活用頂けるよう、多方面からご支援させていただきます。

利用開始は簡単: Cloud Platform を初めてお使い頂く場合でもビジネスにご活用頂き、生産性を上げられるよう、さまざまな種類のリソースが用意されています。ドキュメント、サンプル アーキテクチャ、ソリューション ペーパーには、優れたアプリケーションの作成に役立つ情報が記載されています。

コミュニティへの参加: Cloud Platform ツールを使用して優れたアプリケーションを作成しているデベロッパーのコミュニティに参加しましょう。迷いを払拭することができます。また、Google+ のコミュニティに参加すれば、他の Cloud Platform ユーザーと直接やり取りできるようになります。

プレミアム サポート: Google Cloud Platform では必要に応じて年中無休でいつでもアクセスできる広範囲にわたるサポートをご用意しています。詳しくは、サポート パッケージをご覧ください。

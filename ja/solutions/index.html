ソリューション

Cloud Platform では、さまざまな種類の堅牢なアプリケーションを作成できます。モバイル、ゲーム、Hadoop のソリューションを、いかにスピーディーに、かつ簡単に作成できるかご覧ください。

モバイル

モバイル アプリのバックエンドを作成し、ホストできます。インフラストラクチャが自動で管理されるため、アプリに集中することができます。

モバイルの詳細

Hadoop

Google Compute Engine の仮想マシンでオープンソース Apache Hadoop の速さを実感できます。ジョブ パフォーマンスが高く、課金は分単位なので、過剰な請求が発生する心配はありません。

Hadoop の詳細

ゲーム

Google Cloud Platform なら、基盤となるインフラストラクチャを気にせずに大規模でスケーラブルなゲームを簡単に作成できます。

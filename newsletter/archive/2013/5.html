Sample App: Google Compute Engine Cluster for Hadoop

This is the sample application (command line application), that automatically starts Google Compute Engine instances, sets up Hadoop on them, and runs MapReduce on the Hadoop. The MapReduce job on the Hadoop takes input and output from/to Google Cloud Storage.
Sample App: Mobile Shopping Assistant

Download the Mobile Shopping Assistant sample application (App Engine backend and Android client) to see how the reference architecture has been implemented for a particular scenario. The sample application features a REST API exposed through Google Cloud Endpoints with integrated OAuth-based client authentication, geo-proximity search, push notification, asynchronous request processing, scheduled tasks, and image serving from Google Cloud Storage.
Sample App: Load Balanced Game Server on Google Compute Engine

The sample sets up game servers on Google Compute Engine instances. The instances are managed by App Engine, and new instances are automatically created when load levels of the existing servers are high. The App Engine application automatically selects game servers (Google Compute Engine instance) with low load level for load balancing.

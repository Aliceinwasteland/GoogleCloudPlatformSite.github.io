Dedicated Memcache

Dedicated memcache offers dedicated capacity and ops/s dedicated to a single application. For applications with larger capacity or ops/s requirements, dedicated memcache can increase end user performance and reduce datastore costs by increasing cache hit ratios. We're collecting input from Google Cloud Platform customers on what types of caching solutions they'd like to see. Tell us which features you'd like to see
Console Support for Compute Engine Load Balancing

You can administer load balancing in a more intuitive and less error-prone way in the Cloud Console. To access, click on the "Load balancing" link on the left hand side of your Cloud Console menu.
Google Cloud Storage Server-Side Encryption

Google Cloud Storage now automatically encrypts all data before it is written to disk, at no additional charge. There is no setup or configuration required, no need to modify the way you access the service and no visible performance impact.
Persistent Disk Differential Snapshots

Cloud Platform makes it easy to create a snapshot of a Google Compute Engine persistent disk and keep that snapshot copy stored in Google Cloud Storage for backup, disaster recovery and migration purposes.

Managing Hadoop Clusters on Google Compute Engine

This paper presents several techniques for those who wish to manage their own Hadoop installations on Google Compute Engine. It intends to show how some of the same features that make Compute Engine a powerful platform upon which to run Hadoop, when combined with the gcutil command line utility, can provide the basis for Hadoop cluster management solutions.

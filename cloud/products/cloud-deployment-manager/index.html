Cloud Deployment Manager

Cloud Deployment Manager allows developers to easily design, share, deploy and manage complex Cloud Platform solutions using simple, declarative templates. From simple web servers to complex highly available clusters, Deployment Manager allows teams to spend less time managing, and more time building.

Features

Simple, repeatable deployments in the Cloud

Using Deployment Managerâ€™s declarative, parameterizable templates makes operations on the Cloud Platform simple and repeatable, and frees operations teams from the burden of managing the state of their infrastructure, or managing change.

Let Google look after your infrastructure

Deployment Manager can actively monitor the state of your resources both during and and after a template is deployed. It can reason about many types of Cloud Platform resources - removing the need to wire up many APIs.

Straightforward tools and deep integration

Use our simple scriptable command line interface to manage your deployments, or leverage a fully-featured oAuth 2.0 compliant RESTful API to integrate deeply into your own toolchain and workflows.

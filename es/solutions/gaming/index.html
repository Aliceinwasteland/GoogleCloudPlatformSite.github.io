Soluciones

Publicación de archivos multimedia de gran tamaño

Los usuarios exigen imágenes de alta resolución y descargas de todos los tamaños. Fideliza a los usuarios para que vuelvan a utilizar tu aplicación ofreciendo contenido actualizado regularmente. Google Cloud Storage ofrece una interfaz sencilla basada en REST para almacenar archivos de gran tamaño en la nube y utiliza la misma red de alta velocidad en la que funcionan las aplicaciones de Google.

Análisis de la participación de los usuarios

Entender las acciones de los usuarios y el rendimiento del juego es clave para que tu aplicación tenga valor añadido. Pero, para ello, necesitas algo más que un simple registro: necesitas un análisis potente. Google BigQuery ofrece la misma tecnología que usamos en Google para solucionar estos problemas con las aplicaciones y permite realizar análisis ad hoc en tiempo real de terabytes de datos para incluirlos en tus paneles. Asimismo, puedes utilizar MapReduce en App Engine para fragmentar conjuntos de datos enormes en paralelo para el análisis que no se realiza en tiempo real.

Sincronización del estado del juego en varios dispositivos

Almacena el estado del juego en el almacén de datos de App Engine o en Google Cloud Storage y estará disponible desde todos los dispositivos de los usuarios, además de permitirte obtener información detallada derivada.

Integración sencilla con dispositivos móviles

Utiliza Google Cloud Endpoints para exponer automáticamente la lógica del servidor como API web y generar bibliotecas cliente de tipo estricto para iOS, Android y JavaScript para aplicaciones web estándar.

Creación de una amplia gama de juegos en Google Cloud Platform

Google Cloud Platform te permite crear juegos para plataformas para móviles, como Android e iOS, y plataformas web, como Chrome OS y Facebook. También puedes crear distintos tipos de juegos, desde juegos basados en turnos hasta juegos sociales.

Distribución y obtención de ingresos de tu juego con Google

Obtén ingresos de tu juego con Google AdSense y Pagos integrados en aplicaciones de Google. AdSense te permite mostrar anuncios orientados que concuerden con el aspecto y las funciones del juego. Pagos integrados en aplicaciones te permite vender mercancías virtuales con una tarifa plana del 5%. Distribuye el juego mediante Chrome Web Store para juegos web o Google Play para juegos para móviles.

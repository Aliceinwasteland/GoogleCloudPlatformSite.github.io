Implementa una aplicación en 300 segundos

Desarrollar un backend para móviles es muy sencillo si utilizas código personalizado con Google Cloud Endpoints. Las bibliotecas cliente de tipología estricta generadas automáticamente para iOS, Android y JavaScript facilitan las llamadas a las API del backend.

Ventajas de crear aplicaciones para móviles en Google Cloud Platform:

Escalado automático y distribución de la carga

Lo único que tienes que hacer es escribir el código de la aplicación, del resto nos encargamos nosotros. Sea cual sea el número de usuarios, App Engine ajustará la escala a tus necesidades.

Suministro de contenido estático

App Engine suministra archivos estáticos desde servidores y memorias caché específicos que se encuentran separados de los servidores de aplicaciones.

Procesamiento de datos

Los registros de las aplicaciones pueden proporcionar información sobre su uso y sobre el comportamiento del backend. BigQuery te permite realizar análisis ad hoc en tiempo real de los archivos de registro y analiza terabytes de datos en cuestión de segundos.

Búsqueda por proximidad geográfica

Aprovecha la API de búsqueda de App Engine para realizar búsquedas por proximidad geográfica.

Notificaciones push

Utiliza la mensajería en la nube de Google para enviar datos a los usuarios de Android y la API de sockets para enviar notificaciones push a los usuarios de iOS.

Almacenamiento de datos

Google Cloud Storage te permite almacenar objetos y archivos con tamaños hasta de terabytes mediante una sencilla interfaz basada en REST.

Escalabilidad rápida de Hadoop en Google Cloud Platform

La biblioteca de software Apache Hadoop es un marco que permite procesar de manera distribuida grandes conjuntos de datos entre agrupaciones de ordenadores con modelos de programación sencillos. Está diseñada para aumentar la escala de servidores individuales a miles de máquinas, cada una de ellas con capacidades informáticas y de almacenamiento locales. En lugar de tener que depender del hardware para ofrecer una disponibilidad alta, la biblioteca está diseñada para detectar y gestionar los errores en la capa de aplicación, ofreciendo así un servicio de alta disponibilidad en una agrupación de ordenadores, cada uno de los cuales puede tender a sufrir errores.

MapR Technologies

MapR utilizó Google Compute Engine para establecer un nuevo récord mundial de MinuteSort, ordenando 15.000 millones de registros de 100 bytes (1,5 billones de bytes) en 60 segundos. Esta marca, que se suele conocer como la Copa del Mundo de la ordenación de datos, demostró la rapidez con la que se pueden ordenar los datos que se inician y finalizan en discos. Mientras que el récord anterior de MinuteSort se logró con hardware personalizado, MapR lo estableció con Google Compute Engine, Hadoop MapReduce y la distribución MapR para Apache Hadoop.

Cloud Datastore

Utiliza una base de datos sin esquema NoSQL administrada para almacenar datos no relacionales. Cloud Datastore ajusta la escala automáticamente según tus necesidades y admite transacciones, así como consultas sólidas tipo SQL.

Funciones

Acceso sin esquema y con consultas tipo SQL

Olvídate de los modelos de datos y la migración. Cloud Datastore es un servicio de almacenamiento sin esquema que te permite ser ágil al eliminar la necesidad de pensar en la estructura subyacente de los datos. Cloud Datastore proporciona un motor de consulta sólido que te permite buscar datos en varias propiedades y ordenarlos según convenga.

Base de datos administrada

Cloud Datastore está totalmente administrado. Google se encarga automáticamente de la fragmentación y la replicación para proporcionarte una base de datos coherente y con alta disponibilidad.

Ajuste automático de la escala según los usuarios

Cloud Datastore ajusta automáticamente la escala según tus necesidades. Esto te permite centrarte en crear la aplicación y olvidarte de la administración y la previsión de cargas.

Transacciones ACID

Cloud Datastore proporciona transacciones ACID con control de simultaneidad optimista. Tu aplicación puede ejecutar varias operaciones de almacén de datos en una única transacción, que se ejecutarán correctamente o con errores en su totalidad, garantizando así la integridad de tus datos.

Redundancia integrada

Con una sola llamada a la API, los datos se replican automáticamente entre varios centros de datos. La alta disponibilidad y la durabilidad forman parte de su esencia.

Herramientas locales para desarrolladores

Con el kit de desarrollo de Cloud Datastore puedes desarrollar, probar e iterar las aplicaciones localmente sin tener que realizar implementaciones completas.

Acceso a los datos desde cualquier lugar

Crea soluciones que abarquen App Engine y Compute Engine y confía en Cloud Datastore como punto de integración. Con la interfaz REST que muestra Cloud Datastore, cualquier destino de implementación puede acceder fácilmente a los datos.

Precios

Las tarifas y el nivel gratuito de Google Cloud Datastore son iguales que las tarifas del almacén de datos de App Engine que se detallan a continuación. Ten en cuenta que Cloud Datastore tiene un componente de servidor que se ejecuta en Google App Engine, por lo que habrá costes horarios por instancia.

Recursos

  • Descripción general de Google Cloud Datastore

    Google Cloud Datastore proporciona un conjunto de herramientas y bibliotecas cliente para acceder a Google Cloud Datastore y administrarlo desde clientes que se ejecuten en instancias de Google Cloud Compute Engine u otras ubicaciones. Proporciona una API basada en RPC alojada en el mismo servicio de Google que aloja Google Maps, YouTube y las API de Google Apps. Puedes utilizarla con la mayoría de aplicaciones y lenguajes, por ejemplo, con una aplicación web node.js que se ejecute en una máquina virtual o con un servidor de aplicaciones Java local.

  • Activación de Google Cloud Datastore

    A continuación se describe cómo activar la API de Google Cloud Datastore, las cuotas asociadas con esta API y cómo activar la facturación. El proceso de activación puede variar en función de la configuración y el entorno existentes.

  • Conceptos básicos sobre Cloud Datastore

    Cloud Datastore es un almacén de datos de objetos sin esquema que proporciona almacenamiento sólido y escalable para tu aplicación, con las siguientes funciones: sin periodos de inactividad planeados, transacciones atómicas, alta disponibilidad de lecturas y escrituras, elevada coherencia de lecturas y consultas de antecesor, y coherencia eventual para todas las demás consultas.

Google Prediction API - Predictive Analysis & Prediction Algorithms — Google Cloud Platform

Prediction API

Utiliza los algoritmos de aprendizaje automático de Google para analizar datos y predecir resultados futuros con una interfaz REST conocida.

Pruébalo ahora

Funciones

Datos prácticos

Predice tendencias futuras con datos históricos. Enruta mensajes, detecta spam y recomienda productos a los usuarios mediante Prediction API. La potente infraestructura de Google analiza los datos por ti.

Rapidez y fiabilidad

Los datos se replican en varios centros de datos mediante Cloud Storage. La mayoría de consultas de predicción tardan menos de 200 ms. Hay disponibles opciones de rendimiento superior. Consulta la documentación para obtener consejos de rendimiento.

Integración en la nube

Los servicios de Google están diseñados para colaborar. Utiliza los resultados de Google BigQuery o importa conjuntos de datos de hasta 2,5 GB de Cloud Storage para crear un modelo predictivo. Utiliza Prediction API directamente desde Google App Engine.

Herramientas de desarrollo potentes

Prediction API se puede integrar con App Engine, y la API REST está disponible mediante bibliotecas de muchos lenguajes populares, como Python, JavaScript y .NET. Consulta la documentación para ver la lista completa.

Ejemplos y asistencia

Para obtener más información sobre Prediction API, consulta las aplicaciones de ejemplo. Echa un vistazo a las demostraciones y las aplicaciones destacadas para estimular tus ideas.

Precios flexibles

El uso limitado es gratuito durante los seis primeros meses. Después, el sencillo esquema de precios de Prediction API va acompañado de un acuerdo de nivel de servicio con una disponibilidad del 99,9%. Como en otros productos de Google Cloud, pagas por lo que utilizas.

Precios

Cuota gratuita

El uso es gratuito durante los seis primeros meses, hasta los límites siguientes por proyecto de Google Cloud Console. Esta cuota gratuita se aplica aunque la facturación esté habilitada, hasta la fecha de vencimiento a seis meses.

  • Límites de uso:
    • Predicciones: 100 predicciones/día
    • Predicciones de modelos alojados: los modelos alojados tienen un límite de uso de 100 predicciones/día/usuario en todos los modelos.
    • Entrenamiento: 5 MB entrenados/día
    • Transmisión de actualizaciones: 100 actualizaciones transmitidas/día
  • Límite total: 20.000 predicciones
  • Vencimiento: la cuota gratuita vence seis meses después de activar Google Prediction para el proyecto en Google Cloud Console.

Ten en cuenta que algunos escenarios de uso de Prediction API dependen de Google Cloud Storage, que tiene unas condiciones de precios distintas de las de Prediction API.

Uso de pago

El uso de pago tiene un acuerdo de nivel de servicio con una disponibilidad del 99,9%. Puedes obtener más información sobre el acuerdo de nivel de servicio aquí.

Supervisa el uso en Google Cloud Console. El uso que supere la cuota gratuita no se podrá realizar si no has habilitado la facturación para el proyecto. En cuanto superes la cuota gratuita, se aplicarán los siguientes precios y límites:

  • Tarifa base: tarifa mensual de 10 USD por proyecto de Google Cloud Console
  • Tarifas de uso:
    • Predicción:
      • 10.000 predicciones/mes: 0,00 USD
      • Más de 10.001 predicciones/mes: 0,50 USD/1.000 predicciones después de las 10.000 iniciales
    • Entrenamiento:
      • 0,002 USD/MB entrenados de forma masiva (tamaño máximo de cada conjunto de datos: 2,5 GB)
      • 0-10.000 actualizaciones transmitidas: 0,00 USD
      • Más de 10.001 actualizaciones transmitidas: 0,05 USD/1.000 actualizaciones después de las 10.000 iniciales
  • Límites de uso:
    • Si tienes previsto realizar más de 40.000 predicciones/día, ponte en contacto con nosotros.
    • Google Prediction tiene un límite predeterminado de 2.000.000 predicciones/día por proyecto para tus modelos propios.
    • Los modelos alojados tienen un límite de uso de 100 predicciones/día/usuario en todos los modelos de demostración. Los desarrolladores pueden configurar límites de uso de pago más altos para los modelos que creen. Consulta la documentación de cada modelo para obtener más información.

Importante: Estos cargos no incluyen las tarifas de Google Cloud Storage que podrían requerirse para almacenar los datos de entrenamiento. Consulta los precios de Google Cloud Storage para averiguar cuáles serían los costes adicionales que tendrías que abonar.

Recursos

  • ¿Qué es Google Prediction API?

    Las herramientas de aprendizaje automático basadas en la nube de Google pueden ayudar a analizar los datos para añadir las funciones siguientes a tus aplicaciones: análisis de opinión de clientes, detección de spam, decisiones de enrutamiento de mensajes, análisis de oportunidades de venta complementaria, clasificación de documentos y correos electrónicos, entre otras.

  • Predicción de compras

    Imaginemos un sitio en el que se venda cerveza, vino y queso. Queremos predecir si a un visitante le interesaría comprar vino teniendo en cuenta su historial de compras. En esta situación, podríamos crear datos de entrenamiento con tres elementos: el número de veces que el cliente ha comprado queso de postre gourmet; un valor de 1 si el cliente ha comprado vino alguna vez y de 0 en caso contrario; el número de veces que el cliente ha comprado cerveza.

  • Creación de un modelo de análisis de opinión

    En este documento se explica cómo crear un modelo de análisis de opinión básico mediante Google Prediction API. Un modelo de análisis de opinión se utiliza para analizar una cadena de texto y clasificarla con una de las etiquetas que proporciones; por ejemplo, podrías analizar un tweet para averiguar si es positivo o negativo o analizar un mensaje de correo electrónico para averiguar si en él se expresa felicidad, frustración o tristeza.

Cloud Storage

Utiliza un servicio de almacenamiento de objetos duradero y con alta disponibilidad. Con el almacenamiento en la memoria caché perimetral global, los usuarios podrán acceder rápidamente a los datos de la aplicación desde cualquier ubicación.

Pruébalo ahora

Funciones

Seguridad y protección

Los datos están protegidos mediante el almacenamiento redundante en varias ubicaciones físicas. OAuth y los controles de acceso granular forman una seguridad sólida y que se puede configurar.

Precios competitivos y flexibles

Elige una solución de almacenamiento que se ajusta a la naturaleza de tus datos, tanto si se trata de almacenamiento estándar como si lo que necesitas es almacenamiento de disponibilidad reducida y duradera. Al igual que con los otros servicios de Cloud Platform, solo pagas por lo que utilizas.

Almacenamiento de objetos con una API con todas las funciones

Aunque se puede acceder a Cloud Storage a través de una API sencilla, los conjuntos de funciones avanzados añaden flexibilidad y potencia. Las versiones de objetos, las subidas en paralelo y la comprobación de integridad basada en CRC ofrecen a los desarrolladores la capacidad de crear aplicaciones sólidas y sofisticadas. Puedes acceder a la API directamente mediante XML o JSON o utilizar bibliotecas para varios lenguajes de programación populares.

Acceso flexible

Puedes proporcionar objetos estáticos directamente desde Cloud Storage o utilizarlo como almacén de datos para otros productos de Google. Accede al almacenamiento desde BigQuery, App Engine u otros servicios de Cloud Platform. Se puede acceder a los datos a través de una API REST autenticada que ofrece velocidad, flexibilidad y seguridad respaldada por OAuth.

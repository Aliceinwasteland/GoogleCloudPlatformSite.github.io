Cloud DNS — Google Cloud Platform

Cloud DNS

DNS fiable, resistente y de baja latencia ofrecido por la red mundial de servidores DNS Anycast que Google proporciona. Crea registros DNS con una interfaz de línea de comandos fácil de usar o programa con una API RESTful con todas las funciones para personalizar el servicio según tus necesidades específicas.

Pruébalo ahora

Funciones

Servicio DNS fiable y de alto rendimiento

Utiliza la infraestructura de Google para ofrecer DNS de gran volumen y con calidad de producción. Tus usuarios tendrán un acceso fiable y de baja latencia a la infraestructura de Google desde cualquier lugar del mundo mediante nuestra red de servidores de nombres Anycast.

Fácil de usar, adaptable a tus necesidades

Utiliza nuestra sencilla interfaz de línea de comandos que permite añadir nuevas secuencias de comandos para administrar tus registros. O accede a la API RESTful con todas las funciones para crear tu propia interfaz de DNS personalizada.

Administrar registros de todos los servicios

Conecta los registros DNS para servicios de la plataforma de Google como Compute Engine, App Engine o Google Cloud Storage.

BigQuery

Analiza grandes volúmenes de datos (Big Data) en la nube con BigQuery. Ejecuta consultas tipo SQL rápidas en conjuntos de datos de varios terabytes en cuestión de segundos. Escalable y fácil de utilizar, BigQuery ofrece información detallada en tiempo real sobre los datos.

Funciones

Todo en segundo plano

Las consultas se pueden ejecutar de manera asíncrona en segundo plano y su estado se puede sondear. Accede al historial de tareas y consultas junto al resto de recursos de Cloud Platform en Google Cloud Console.

Importación sencilla de los datos

Carga los datos de forma masiva mediante Google Cloud Storage o transmítelos en ráfagas de hasta 1.000 filas por segundo. Puedes afrontar tareas de registro o grabación de transacciones de alto volumen en tiempo real y los datos están disponibles para realizar análisis rápidamente y de manera sencilla.

Grandes volúmenes de datos asequibles

Cuando hablamos de "Big Data", hablamos en serio. Los primeros 100 GB de datos procesados cada mes son gratis. Almacenaremos y consultaremos miles de terabytes por ti. Nuestra estructura de precios es sencilla y transparente, y te damos toda la visibilidad y el control. Solo pagas por el almacenamiento que necesitas y las consultas que ejecutas.

La interfaz correcta

Con las distintas interfaces para la administración y los desarrolladores, tendrás acceso a las herramientas que necesitas. Controla el acceso a nivel de proyecto y de conjunto de datos con la Consola de API de Google. BigQuery ayuda a equipos de todo tamaño a trabajar tal como desean.

Precios

BigQuery usa una estructura de datos por columnas, lo que significa que, para una consulta determinada, solo se paga por los datos procesados en cada columna y no en toda la tabla. Los primeros 100 GB de datos procesados por mes no se cobran.

Los precios entraron en vigor el 1 de abril de 2014.

Recursos

  • ¿Qué es BigQuery?

    La consulta de conjuntos de datos de gran envergadura puede llevar mucho tiempo y salir muy cara sin el hardware y la infraestructura adecuados. Google BigQuery resuelve este problema al permitir realizar consultas tipo SQL muy rápidas en tablas de solo anexión con la potencia de procesamiento que ofrece la infraestructura de Google. Traslada tus datos a BigQuery y deja que nos encarguemos de la parte difícil.

  • Carga de datos en BigQuery

    Para poder consultar los datos, primero tienes que cargarlos en BigQuery. Puedes cargar los datos de forma masiva mediante una tarea o transmitir registros individualmente.

  • Consulta de datos

    Las consultas se escriben en el dialecto SQL de BigQuery. BigQuery admite tanto el método de consulta síncrona como el método de consulta asíncrona. Ambos métodos están controlados por una tarea, pero el método síncrono incluye un valor de tiempo de espera que espera hasta que la tarea haya finalizado para ofrecer un resultado.

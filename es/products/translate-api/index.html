Translate API

Traduce de manera rápida y dinámica entre miles de pares de idiomas disponibles en tu aplicación gracias a la integración con Traductor de Google.

Pruébalo ahora

Funciones

Acceso dinámico a idiomas

Google Translate API admite muchos idiomas. Puedes detectar el idioma de un documento y traducirlo mediante una API REST.

Accesible con API de Google

Se puede acceder a Translate API mediante la API de Google estándar. Consulta las bibliotecas y el código de ejemplo para diez lenguajes de programación, incluidos Python, Objective C y Ruby.

Precios asequibles y comprensibles

Los precios de Translate API se basan en los millones de caracteres que se envían. Nuestro esquema de precios es fácil de entender y muy asequible.

Precios

Los precios de Google Translate API se basan en el uso. El uso de la traducción se calcula en millones de caracteres (M), donde 1 M = 106 caracteres. Para obtener más información, consulta Preguntas frecuentes sobre precios.

Importante: Google Translate API v2 requiere datos de facturación para todas las cuentas para poder empezar a utilizar el servicio. Consulta las instrucciones a continuación para habilitar la facturación.

  • Tarifas de uso:
    • Traducción:
      • 20 $ por 1 M de caracteres de texto; los cargos se ajustan en proporción al número de caracteres que se proporcionan en realidad. Por ejemplo, si traduces 500.000 caracteres, se te facturarán 10 $.
    • Detección de idioma:
      • 20 $ por 1 M de caracteres de texto; los cargos se ajustan en proporción al número de caracteres que se proporcionan en realidad.
  • Límites de uso:
    • Google Translate API tiene un límite predeterminado de dos millones de caracteres al día. Puedes aumentar este límite hasta 50 millones de caracteres al día en Cloud Console si sigues las instrucciones que se incluyen a continuación.
    • Si necesitas traducir más de 50 millones de caracteres al día, ponte en contacto con nosotros.

Para habilitar la facturación para tu proyecto, debes hacer lo siguiente:

  1. Ve a Google Cloud Console.
  2. Selecciona un proyecto.
  3. En la barra lateral de la izquierda, selecciona Facturación.
  4. En la parte superior de la página, selecciona Habilitar la facturación.
  5. Rellena el formulario.

Para ver o cambiar los límites de uso de tu proyecto o para solicitar un aumento del límite total, debes hacer lo siguiente:

  1. Ve a Google Cloud Console.
  2. Selecciona un proyecto.
  3. En la barra lateral de la izquierda, selecciona APIs y autenticación y, a continuación, selecciona una API.
  4. En la página de información de la API, selecciona el enlace Cuota ubicado cerca del nombre de la API.

Productos

Ejecuta tu aplicación con la misma tecnología y las mismas herramientas que se utilizan en Google.
Cloud Platform proporciona los elementos necesarios para desarrollar aplicaciones rápidamente, con todos los servicios que necesitas.

Computación

Compute Engine

Compute Engine es la infraestructura como servicio (IaaS, del inglés Infrastructure-as-a-Service) de Google. Ejecuta cargas de trabajo a gran escala en máquinas virtuales alojadas en la infraestructura de Google. Elige una máquina virtual que se ajuste a tus necesidades y disfruta del rendimiento y la coherencia que ofrece la red de fibra óptica mundial de Google. Con la facturación por minuto, solo pagarás por lo que utilices.

App Engine

App Engine es la plataforma como servicio (PaaS, del inglés Platform-as-a-Service) de Google. Desarrolla aplicaciones fácilmente con servicios integrados que mejoran tu productividad. Impleméntalas en una plataforma totalmente administrada y deja que Google se encargue del resto. Para empezar a crear aplicaciones de inmediato, descarga el SDK. Es gratis y no necesitas tarjeta de crédito.

Almacenamiento

Cloud SQL

Almacena y administra los datos con una base de datos MySQL relacional totalmente administrada. Google se encarga de la replicación, la gestión de parches y la administración de la base de datos para garantizar su disponibilidad y rendimiento.

Cloud Storage

Utiliza un servicio de almacenamiento de objetos duradero y con alta disponibilidad. Con el almacenamiento en la memoria caché perimetral global, los usuarios podrán acceder rápidamente a los datos de la aplicación desde cualquier ubicación. Google administra el control de versiones, garantiza un acuerdo de nivel de servicio sólido y proporciona una API sencilla con la que podrás administrar los datos mediante programación.

Cloud Datastore

Cloud Datastore proporciona una base de datos sin esquema NoSQL administrada para almacenar datos no relacionales. Cloud Datastore ajusta la escala automáticamente según tus necesidades y admite transacciones, así como consultas sólidas tipo SQL.

Big Data

BigQuery

Analiza grandes volúmenes de datos (Big Data) en la nube con BigQuery. Ejecuta consultas tipo SQL rápidas en conjuntos de datos de varios terabytes en cuestión de segundos. Escalable y fácil de utilizar, BigQuery ofrece información detallada en tiempo real sobre los datos.

Servicios

DNS de Google Cloud

DNS fiable, resistente y de baja latencia ofrecido por la red mundial de servidores DNS Anycast que Google proporciona. Crea registros DNS con una interfaz de línea de comandos fácil de usar o programa con una API RESTful con todas las funciones para personalizar el servicio según tus necesidades específicas.

Cloud Endpoints

Crea servicios REST a partir del código y haz que sean accesibles desde clientes iOS, Android y JavaScript. Genera automáticamente bibliotecas cliente para que la interfaz sea fácil de configurar. La infraestructura esencial integrada incluye protección contra ataques de denegación de servicio, compatibilidad con OAuth 2.0 y administración de claves de clientes.

Administración

Gestor de implementación en la nube

El Gestor de implementación en la nube permite a los desarrolladores diseñar, compartir, implementar y gestionar soluciones complejas de Cloud Platform mediante plantillas sencillas y declarativas. Desde servidores web sencillos hasta agrupaciones complejas de alta disponibilidad, el Gestor de implementación permite a los equipos dedicar menos tiempo a tareas de gestión y más tiempo a la creación.

Herramientas para desarrolladores

SDK de Google Cloud

El SDK de Google Cloud contiene herramientas y bibliotecas que te permiten crear y gestionar los recursos de Google Cloud Platform como, por ejemplo, App Engine, Compute Engine, Cloud Storage, Cloud SQL y BigQuery.

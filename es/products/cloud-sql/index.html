Google Cloud SQL - SQL Server, Cloud Database & Online Database — Google Cloud Platform

Cloud SQL

Almacena y administra los datos con una base de datos MySQL relacional totalmente administrada. Google se encarga de la replicación, la gestión de parches y la administración de la base de datos para garantizar su disponibilidad y rendimiento.

Pruébalo ahora

Funciones

Implementa bases de datos MySQL en la nube sin quebraderos de cabeza. Google Cloud Platform proporciona bases de datos potentes que se ejecutan con rapidez, no se quedan sin espacio y dan a tu aplicación el almacenamiento fiable y redundante que necesita.

Infraestructura conocida

Puedes crear e implementar aplicaciones para la nube más rápido porque Cloud SQL ofrece bases de datos MySQL estándar, las bases de datos de código abierto más populares del mundo. Las instancias están disponibles con hasta 16 GB de RAM y 100 GB de capacidad de almacenamiento.

Cargos flexibles

Gracias a la opción de pago por uso, empezar es económico. Si ejecutas una base de datos que se utiliza poco o esporádicamente, ahorrarás dinero si pagas solo por el tiempo durante el que accedes a los datos. La opción por paquetes te permite controlar los costes de las instancias que soportan una carga mayor.

Seguridad, disponibilidad, durabilidad

Your data is replicated in many geographic locations as standard, and failover between them is handled automatically. This means your data is safe and your database is available even in the event of a major failure. We also manage your backups, making it easy for you to restore when needed, including point-in-time recovery.

Migración más sencilla y sin bloqueos

Las conexiones y herramientas estándar, como mysqldump, MySQL Wire Protocol y JDBC facilitan la migración a (y desde) Google Cloud Platform y evitan bloqueos.

Control

It's easy to manage and access your instances through a web Console or a command-line interface.

Completamente administrado

No te preocupes por la replicación o por la administración de parches o de bases de datos: nosotros nos ocupamos de ello.

Precios

Google ofrece dos planes de facturación para Cloud SQL:

  • Para desarrolladores con más tráfico, tenemos varios paquetes que ofrecen un descuento y ayudan a predecir los costes de antemano.
  • Para desarrolladores con aplicaciones con poco tráfico, ofrecemos un esquema de precios flexible por uso. Solo se paga por el tiempo que se accede a los datos. Empieza con una base de datos MySQL alojada en la nube por aproximadamente 1 USD al mes.

  • Acerca de Google Cloud SQL

    Google Cloud SQL es una base de datos MySQL ubicada en la nube de Google. Tiene todas las características y funciones de MySQL, con unas cuantas funciones más y algunas funciones no compatibles, como se explica más abajo. Google Cloud SQL es fácil de utilizar, no requiere instalación ni mantenimiento de software y es idóneo para aplicaciones pequeñas y medianas.

Google Cloud Platform으로
모바일 앱 제작

모든 모바일 앱을 위한 백엔드를 구축하고 호스팅해 보세요. 인프라는 자동으로 관리되므로 앱 제작에 집중할 수 있습니다. 또한 수백만 명의 사용자를 지원하도록 무한 확장이 가능합니다.

지금 사용해 보기 또는 영업 팀에 문의하기

300초 안에 앱 배포 가능

Google Cloud Endpoints의 맞춤 코드로 모바일 백엔드를 개발하는 것이 얼마나 간편한지 확인해 보세요. 강력하게 유형이 지정된 iOS, Android 및 자바스크립트용 클라이언트 라이브러리가 자동 생성되어 간편하게 백엔드 API를 호출합니다.

Google Cloud Platform에서 모바일 앱 제작의 이점:

자동 확장 및 부하 분산

애플리케이션 코드를 작성하기만 하면 나머지는 Google에서 처리합니다. 사용자 수에 관계없이 App Engine에서 필요한 만큼 확장합니다.

정적 콘텐츠 제공

App Engine은 애플리케이션 서버와는 별도의 전용 서버 및 캐시에서 정적 파일을 제공합니다.

데이터 처리

애플리케이션 로그는 앱 사용량과 백엔드 동작에 대한 통계를 제공합니다. BigQuery를 사용하면 몇 초 만에 테라바이트 단위의 데이터를 분석하여 로그 파일의 실시간 즉시 분석 통계를 수집할 수 있습니다.

가까운 지역 검색

App Engine의 Search API를 활용하여 가까운 지역을 검색할 수 있습니다.

푸시 알림

Google Cloud 메시징으로 Android 사용자에게 데이터를 전송하고, Sockets API를 사용하여 iOS 사용자에게 알림을 푸시할 수 있습니다.

데이터 저장소

Google Cloud Storage를 사용하면 간단한 REST 기반 인터페이스를 사용하여 최대 테라바이트 단위로 개체와 파일을 저장할 수 있습니다.

시작하려면

모바일 앱을 배포해 보세요. 코드, 가이드 및 Google Cloud Platform만 있으면 됩니다.

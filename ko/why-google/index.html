Benefits of Using Google Cloud Platform — Google Cloud Platform

Google Cloud Platform을 사용해야 하는 이유

Google Cloud Platform을 사용하면 개발자는 뛰어난 확장성과 안정성을 제공하는 Google의 인프라를 기반으로 애플리케이션을 제작하고 테스트한 후 배포할 수 있습니다. 웹, 모바일, 백엔드 솔루션에 맞는 컴퓨팅, 저장소 및 애플리케이션 서비스를 선택해 보세요.

Google의 인프라에서 실행

몇 밀리초 안에 수십억 개의 검색결과를 반환하고, 월 60억 시간 분량의 YouTube 동영상을 제공하며, 4억 2천 5백만 명의 Gmail 사용자에게 저장소를 제공하고 있는 Google의 인프라를 그대로 활용하여 애플리케이션을 제작할 수 있습니다.

글로벌 네트워크: Google에는 가장 광범위하고 발전된 형태의 컴퓨터 네트워크를 보유하고 있습니다. 수천 킬로미터 길이의 광섬유 케이블로 이루어진 Google의 백본 네트워크는 소프트웨어로 정의된 고급 네트워킹을 사용하며, 우수한 확장 가능성과 일관성을 제공하는 에지 캐싱 서비스를 지원합니다. 실제로 Google은 태평양 아래까지 자체 광섬유 케이블을 설치하기도 했습니다.

중복성: 전 세계의 여러 인터넷 접속 포인트에서 강력한 중복성을 제공합니다. 데이터는 복수의 위치에 있는 저장 기기에 자동으로 미러링됩니다.

최첨단 컴퓨터 과학: 인프라 혁신이 하드웨어에서만 진행되는 것은 아닙니다. Google은 MapReduce, BigTable, Dremel과 같은 소프트웨어 인프라 혁신으로 업계를 주도해 왔으며, 현재 Spanner 및 Flume과 같은 제품으로 차세대 혁신을 추진하고 있습니다. Cloud Platform에서 제작하면 Google의 기술 혁신을 더욱 빠르게 접할 수 있습니다.

"서버 및 광섬유 케이블의 거대한 네트워크를 빠르고 효율적으로 구축, 구성 및 운영하는 [Google]의 능력은 물리적인 수치들을 뒤흔들고 있습니다...이러한 능력이 Google을 Google답게 만듭니다. 물리적 네트워크, 수천 킬로미터에 이르는 광섬유 및 수천 개의 서버를 합쳐 놓으면 결국 모든 클라우드의 모태가 되는 것입니다."

제품에 집중

시스템 관리에 대해 염려할 필요 없이 애플리케이션을 빠르게 개발, 배포, 개선할 수 있습니다. Google에서 애플리케이션, 데이터베이스, 저장소 서버를 관리하므로 별도의 관리가 필요하지 않습니다.

관리되는 서비스: 데이터베이스 관리, 서버 구성, 분할 및 부하 분산은 Google에서 담당하니 여러분은 코드에만 집중하면 됩니다. 긴급 상황을 위해 대기하거나 상용구 코드를 작성할 필요가 없습니다.

개발자 도구 및 SDK: Google에서는 Eclipse와 같은 익숙한 개발 도구와 통합되고, API 클라이언트 라이브러리와 명령줄 인터페이스를 제공하므로 원하는 방식으로 쉽게 제작할 수 있습니다.

콘솔 및 관리: 하나의 콘솔에서 모든 애플리케이션을 확인하고 관리할 수 있습니다. 단순한 인터페이스로 애플리케이션의 성능을 확인하고, 계정 및 결제를 관리해 보세요.

조합형 서비스

가상 머신, 관리되는 플랫폼, Blob 저장소, 블록 저장소, NoSQL 데이터 저장소, MySQL 데이터베이스, 빅 데이터 분석 등 Google Cloud Platform에는 애플리케이션 아키텍처에 필요한 모든 서비스가 있습니다.

컴퓨팅: Cloud Platform은 전체적으로 관리되는 플랫폼과 유연한 가상 머신을 모두 제공하므로 요구사항에 맞춰 시스템을 선택할 수 있습니다. 패치나 유지관리에 대한 걱정 없이 코드에만 신경 쓰고 싶다면 Google의 서비스로서의 플랫폼인 App Engine을 사용해 보세요. Compute Engine으로 기초 가상 머신에 액세스하여 필요한 기능을 자유롭게 만들 수 있습니다.

저장소: Google Cloud Platform은 유지관리가 용이하고 빠른 데이터 액세스가 가능한 다양한 저장소 서비스를 제공합니다. Cloud SQL 및 Datastore를 사용하면 MySQL, 또는 스키마가 없는 NoSQL 데이터베이스가 제공되고, Cloud Storage를 사용하면 글로벌 에지 캐싱이 포함된 유연한 개체 저장소가 제공됩니다.

서비스: Google API와 서비스를 사용하여 애플리케이션에서 광범위한 기능을 빠르게 구현해 보세요. 처음부터 끝까지 일일이 제작할 필요 없이 Cloud Platform에서 제공하는 간편한 통합 기능을 활용하기만 하면 됩니다.

수백만 명의 사용자로 확장 및 확장 이후 축소

Cloud Platform에서 호스팅되는 애플리케이션은 부하가 가장 높은 작업을 처리하기 위해 자동으로 확장되고 트래픽이 줄어들면 축소도 가능합니다. 비용은 사용한 만큼만 지불하면 됩니다.

확장: Cloud Platform은 Google의 자체 제품들처럼 트래픽이 급격하게 상승하는 경우에도 확장 가능하도록 설계되었습니다. App Engine 또는 Cloud Datastore와 같이 관리되는 서비스들은 사용자 수에 따라 애플리케이션이 함께 커지도록 하는 자동 확장 기능을 제공합니다.

축소: Cloud Platform에서 확장이 가능하듯이 축소도 가능하기 때문에 필요하지 않은 컴퓨팅 리소스에 대해서는 비용을 지불하지 않아도 됩니다.

신뢰할 수 있는 성능

밀리초의 지연 시간도 문제가 될 수 있습니다. Google의 컴퓨팅 인프라는 일관성 있는 CPU, 메모리 및 디스크 성능을 제공하고, Google의 네트워크와 에지 캐시는 전 세계 사용자에게 신속한 응답을 제공합니다.

CPU, 메모리, 디스크: Google Cloud Platform은 컴퓨팅, 저장소 및 애플리케이션 서비스 범위에서 빠르고 일관성 있는 성능을 제공합니다. 강력한 처리 능력으로 필요한 메모리에 액세스하고, 높은 IOPS로 애플리케이션에서 일관성 있는 성능을 제공합니다. 지연 시간 감소의 혜택을 누리면서 시끄러운 서버 소음도 피할 수 있습니다.

글로벌 네트워크: Google은 소프트웨어로 정의된 네트워킹 기술을 사용하여 전 세계에서 패킷을 라우팅하고, 사용자에게 제공되어야 하는 위치로 데이터가 전송되도록 고속 에지 캐싱을 사용합니다. 밀리초의 지연 시간도 문제가 될 수 있는 경우에는 콘텐츠가 빠르게 전송되는지 확인합니다.

투명한 유지관리: 새롭게 내장된 실시간 마이그레이션 기술을 사용하므로 예약된 유지관리로 인해 가상 머신이 중단되는 일이 발생하지 않습니다. 호스트가 패치되고 데이터 센터가 유지관리되어 안심할 수 있으며, 중단에 대한 걱정이 없습니다.

Sebastian Stadil

Scalr CEO

"Google Cloud Platform은 지금까지 본 플랫폼 중에서 가장 일관성 있는 성능을 제공합니다. 모든 VM과 디스크가 정확히 예상대로 작동하기 때문에 빠르고 지연 시간이 적은 애플리케이션을 제작할 수 있었습니다."

필요한 지원 받기

전 세계 사용자 커뮤니티, 파트너 환경 및 프리미엄 지원 패키지가 제공되며 시작과 성장을 위한 모든 범위의 리소스를 제공합니다.

빠르게 시작하기: Cloud Platform을 처음 사용하는 경우 생산적으로 작업할 수 있도록 도와주는 다양한 리소스가 준비되어 있습니다. 문서, 샘플 아키텍처 및 솔루션 문서에서는 뛰어난 애플리케이션을 제작하기 위한 기본 자료를 제공합니다.

커뮤니티 참여: Cloud Platform 도구를 사용하여 우수한 애플리케이션을 제작하는 개발자들의 커뮤니티에 참여해 보세요. Stack Overflow에서 빠르게 답변을 얻거나 Cloud Platform의 다른 사용자와 직접 교류할 수 있는 Google+의 커뮤니티에 참여할 수 있습니다.

프리미엄 지원: Google Cloud Platform에서는 연중무휴 24시간 문의 서비스가 포함된 다양한 지원 옵션을 제공하고 있습니다. 여러 지원 패키지에 대해 자세히 알아보세요.

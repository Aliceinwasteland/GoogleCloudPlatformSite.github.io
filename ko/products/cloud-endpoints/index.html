Cloud Endpoints — Google Cloud Platform

Cloud Endpoints

RESTful 서비스를 만들고 iOS, Android 및 자바스크립트 고객이 이 서비스에 액세스할 수 있도록 해보세요. 프론트엔드를 쉽게 연결할 수 있도록 클라이언트 라이브러리가 자동으로 생성되고, 서비스 거부 보호, OAuth 2.0 지원 및 클라이언트 키 관리가 내장 기능으로 포함되어 있습니다.

기능

하나의 도구, 여러 클라이언트

소스 하나로 Android, iOS, 웹 기반 클라이언트를 위한 클라이언트 라이브러리를 제작할 수 있습니다. Cloud Endpoints에서는 코드를 래핑하여 몇 단계 만에 API 서버 구축이 가능합니다. Cloud Endpoint API 라이브러리는 자바, Python, Go 및 PHP로 제공됩니다. Google 문서에서 Cloud Endpoints에 대해 자세히 알아보거나 샘플 Tic Tac Toe 웹 애플리케이션을 사용해 보세요.

App Engine 인프라 확장

App Engine에서 사용 가능한 모든 도구와 라이브러리를 이제 휴대기기에서도 사용할 수 있습니다. 추가 작업 없이 App Engine 백엔드를 사용하여 Datastore, Cloud Storage 및 작업 대기열에 액세스해 보세요. Google이 마련한 예를 따라 OAuth 2.0 인증을 빠르게 통합할 수 있습니다.

유지관리가 필요없는 클라이언트 서버

Cloud Endpoints는 App Engine을 사용하므로 유지관리해야 할 서버가 없고, 부하 분산에 대해 염려할 필요가 없으며, 동일한 방식으로 빠르고 쉽게 확장이 가능합니다. App Engine 및 다른 Google Cloud 서비스와 마찬가지로 사용한 만큼만 비용을 지불하면 됩니다.

유연한 클라이언트측 통합

서버측 API에 주석을 추가하고 자동으로 클라이언트 라이브러리를 제작할 수 있습니다. 클라이언트 라이브러리는 Android 및 iOS용으로 제작됩니다. 표준 웹 클라이언트를 준비하고 최소한의 자바스크립트 클라이언트 라이브러리로 실행해 보세요. 모든 클라이언트에서 유사한 API와 동일한 백엔드를 사용하므로 개발 시간이 줄어듭니다.

가격 책정

Cloud Endpoints는 App Engine 및 Compute Engine에서 실행되는 애플리케이션에 무료로 제공됩니다.

문서 및 리소스

Cloud Endpoints를 사용하여 App Engine 애플리케이션용 클라이언트 API를 자동으로 제작해 보세요. 다양한 App Engine 실행을 위해 가이드와 안내가 제공되므로 빠르게 준비하고 실행할 수 있습니다.

리소스

  • Endpoints API 만들기

    Endpoints API는 외부 클라이언트에 액세스할 수 있는 원격 메소드를 제공하는 RPC(원격 절차 호출) 서비스입니다. 각 Endpoints API는 ProtoRPC remote.Service 클래스 및 하나 이상의 메소드를 하위 클래스로 지정하는 RPC 서비스로 구성됩니다.

  • 클라이언트 라이브러리 생성

    엔드포인트를 만든 후 Android 및 iOS용 클라이언트 라이브러리를 생성할 수 있습니다.

  • iOS 클라이언트에서 엔드포인트 사용

    iOS 앱에서 엔드포인트를 사용하는 방법은 다음과 같습니다. 클라이언트 라이브러리 생성기를 컴파일하고 라이브러리를 생성합니다. iOS 프로젝트에 필수 파일을 추가하고, 요청하는 데 사용할 서비스 개체를 만듭니다. 엔드포인트에 의해 노출되는 API를 호출하고, 원한다면 인증된 호출을 위한 지원을 추가합니다.

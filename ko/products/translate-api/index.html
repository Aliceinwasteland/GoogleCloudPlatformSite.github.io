Translate API

Google 번역과의 통합을 통해, 앱 내에서 제공되는 수천 개의 언어 조합으로 신속한 번역이 가능합니다.

기능

동적으로 언어 액세스

Google Translate API는 수십 개의 언어를 지원합니다. RESTful API를 사용하여 문서의 언어를 감지하고 번역할 수 있습니다.

Google API로 액세스 가능

Translate API는 표준 Google API를 통해 액세스할 수 있습니다. Python, Objective C, Ruby 등 10개의 다른 프로그래밍 언어로 작성된 샘플 코드 및 라이브러리를 참조하세요.

저렴하고 간단한 가격 책정 방식

Translate API의 가격은 전송된 영문 100만 자 단위로 책정됩니다. Google의 가격 책정 방식은 이해하기 쉽고 합리적인 가격을 제공합니다.

가격 책정

Google Translate API의 가격은 사용량에 따라 책정됩니다. 번역 사용량은 영문 100만 자 단위(M)로 계산되며 1M은 영문 기준 106자입니다. 자세한 내용은 가격 책정 FAQ를 참조하세요.

중요: Google Translate API 버전 2 서비스를 사용하려면 모든 계정의 결제 정보가 필요합니다. 결제를 사용 설정하는 방법은 아래 안내를 참조하세요.

  • 사용 요금:
    • 번역:
      • 텍스트 영문 100만 자당 $20가 청구되며, 요금은 실제로 제공된 글자수에 비례하여 조정됩니다. 예를 들어 영문 50만 자를 번역한 경우 $10가 청구됩니다.
    • 언어 감지:
      • 텍스트 영문 100만 자당 $20가 청구되며, 요금은 실제로 입력된 글자수에 비례하여 조정됩니다.
  • 사용량 한도:
    • Google Translate API에는 일일 영문 200만 자의 기본 제한이 있습니다. 아래 안내에 따라 Cloud Console에서 이 제한을 일일 영문 5천만 자로 늘릴 수 있습니다.
    • 일일 영문 5천만 자 이상의 번역이 필요한 경우에는 문의해 주세요.

프로젝트에 결제를 사용 설정하려면 다음을 수행합니다.

  1. Google Cloud Console로 이동합니다.
  2. 프로젝트를 선택합니다.
  3. 왼쪽의 사이드바에서 결제을 선택합니다.
  4. 페이지 상단에서 결제 사용 설정을 선택합니다.
  5. 양식을 작성합니다.

프로젝트의 사용량 제한을 확인 또는 변경하거나, 총 제한량 증가를 요청하려면 다음을 수행합니다.

  1. Google Cloud Console로 이동합니다.
  2. 프로젝트를 선택합니다.
  3. 왼쪽 사이드바에서 API 및 인증을 선택한 다음 API를 선택합니다.
  4. API 정보 페이지에서 API 이름 옆에 있는 할당량 링크를 선택합니다.

Cloud Deployment Manager

Cloud Deployment Manager allows developers to easily design, share, deploy and manage complex Cloud Platform solutions using simple, declerative templates. From simple web servers to complex highly available clusters, Deployment Manager allows teams to spend less time managing, and more time building.

기능

클라우드 상의 단순하고 반복적인 배포

배포 관리자가 보유한 매개변수화 가능한 선언적 템플릿을 사용하면 Cloud Platform에서의 작업을 간단하게 반복적으로 수행할 수 있으며 운영팀은 인프라 상태 또는 변경사항을 관리해야 하는 부담에서 벗어날 수 있습니다.

Google을 통한 인프라 관리

배포 관리자는 템플릿을 배포하는 동안이나 배포 이후에 리소스 상태를 적극적으로 모니터링할 수 있습니다. 또한 여러 Cloud Platform 리소스 유형에 대해 논리적으로 판단할 수 있어 많은 API를 연결할 필요가 없어집니다.

손쉬운 도구와 긴밀한 통합

스크립트 가능한 단순한 명령줄 인터페이스를 사용하여 배포를 관리하거나 oAuth 2.0을 준수하는 RESTful API를 사용하여 자신만의 도구 모음 및 워크플로우로 완벽하게 통합이 가능합니다.

리소스

  • 빠른 시작: nginx 및 PHP 서버 시작

    nginx는 오픈 소스 기반의 고성능 웹 서버입니다. 샘플 템플릿은 복제된 구성을 선택적으로 배포하고 기본형 PHP를 지원합니다. 본 가이드에서는 서버를 설명하는 템플릿을 설치하고 이를 사용하여 프로젝트 내에서 배포를 수행하는 과정을 안내합니다.

  • 빠른 시작 : node.js 서버 시작

    node.js는 오픈 소스 기반의 자바스크립트 웹 서버입니다. 샘플 템플릿은 복제된 구성을 선택적으로 배포합니다. 본 가이드에서는 서버를 설명하는 템플릿을 설치하고 이를 사용하여 프로젝트 내에서 배포를 수행하는 과정을 안내합니다.

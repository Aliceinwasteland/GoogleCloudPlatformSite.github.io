Products — Google Cloud Platform

제품

Google 고유의 기술과 도구를 기반으로 애플리케이션을 실행할 수 있습니다.
Cloud Platform에서 기본적인 요소를 제공하므로 필요한 서비스를 사용하여 신속한 개발이 가능합니다.

컴퓨팅

App Engine

App Engine은 Google의 PaaS(서비스로서의 플랫폼)입니다. 생산성 높은 내장형 서비스를 사용하여 쉽게 애플리케이션을 개발할 수 있고, 완전히 관리되는 플랫폼을 통해 배포하면 됩니다. 나머지 작업은 Google에서 처리합니다. 신용카드를 사용할 필요 없이 무료로 SDK를 다운로드하고 곧바로 제작을 시작할 수 있습니다.

  • 기능
  • 사례 연구
  • 가격 책정
  • 도움말

Compute Engine

Compute Engine은 Google의 IaaS(서비스로서의 인프라)입니다. Google 인프라에서 호스팅되는 가상 머신에서 대규모 작업을 실행해 보세요. 필요에 따라 원하는 VM을 선택할 수 있으며 Google의 글로벌 광섬유 네트워크의 일관된 성능을 활용할 수 있습니다. 분당 과금제로, 사용한 만큼만 비용을 지불하면 됩니다.

  • 사례 연구
  • 가격 책정
  • 도움말

저장소

Cloud Datastore

Cloud Datastore는 비관계형 데이터를 저장하는 스키마 없는 관리형 NoSQL 데이터베이스를 제공합니다. Cloud Datastore는 필요한 만큼 자동으로 크기를 조절하고 트랜잭션은 물론 SQL과 비슷한 강력한 쿼리를 지원합니다.

  • 기능
  • 가격 책정
  • 도움말

Cloud SQL

완전히 관리되는 관계형 MySQL 데이터베이스를 사용하여 데이터를 저장하고 관리해 보세요. Google에서 복제, 패치 관리 및 데이터베이스 관리를 처리하기 때문에 가용성과 성능이 보장됩니다.

  • 기능
  • 사례 연구
  • 가격 책정
  • 도움말

Cloud Storage

뛰어난 내구성과 가용성을 제공하는 개체 저장 서비스를 사용해 보세요. 글로벌 에지 캐싱으로 위치에 상관없이 어디서나 앱의 데이터에 빠르게 액세스할 수 있습니다. 버전 관리는 Google에서 수행하고, 강력한 SLA도 보장됩니다. 또한 프로그래밍 방식으로 데이터를 관리할 수 있는 간단한 API도 제공합니다.

  • 기능
  • 사례 연구
  • 가격 책정
  • 도움말

빅 데이터

BigQuery

BigQuery로 클라우드에서 빅 데이터를 분석해 보세요. 테라바이트급 데이터 세트에서 SQL과 비슷한 쿼리를 단 몇 초 만에 빠르게 실행할 수 있습니다. 확장 가능하고 사용이 용이한 BigQuery는 데이터에 대한 실시간 통계를 제공합니다.

  • 기능
  • 사례 연구
  • 가격 책정
  • 도움말

서비스

Cloud DNS

전 세계에 존재하는 Google의 Anycast DNS 서버 네트워크에서 제공하는 안정적이고 응답성이 높으며 지연 시간이 적은 DNS 서비스입니다. 사용이 간편한 명령줄 인터페이스로 DNS 레코드를 생성하거나 모든 기능을 갖춘 RESTful API로 프로그램을 작성하여 특정 요구사항에 맞도록 서비스를 맞춤설정할 수 있습니다.

  • 기능
  • 가격 책정
  • 도움말

Prediction API

Google의 기계 학습 알고리즘을 사용하여 익숙한 RESTful 인터페이스에서 데이터를 분석하고 향후 결과를 예측해 보세요.

  • 기능
  • 가격 책정
  • 도움말

Cloud Endpoints

코드로 RESTful 서비스를 만들고 iOS, Android 및 자바스크립트 고객이 이 서비스에 액세스할 수 있도록 해보세요. 프론트엔드를 쉽게 연결할 수 있도록 클라이언트 라이브러리가 자동으로 생성되고, 서비스 거부 보호, OAuth 2.0 지원, 클라이언트 키 관리가 내장된 중요 인프라가 포함되어 있습니다.

  • 기능
  • 사례 연구
  • 가격 책정
  • 도움말

Translate API

다국어 앱을 만들고 프로그래밍 방식으로 텍스트를 다른 언어로 번역해 보세요. 수천 개의 언어 조합을 사용할 수 있습니다.

  • 기능
  • 가격 책정
  • 도움말

관리

Cloud Deployment Manager

개발자는 Cloud Deployment Manager의 단순한 선언적 템플릿을 사용하여 복잡한 Cloud Platform 솔루션을 손쉽게 설계, 공유, 배포, 관리할 수 있습니다. 간단한 웹 서버에서부터 복잡하고 가용성이 높은 클러스터까지 배포 관리자는 팀이 관리보다 구축에 더 많은 시간을 보낼 수 있도록 해줍니다.

개발자 도구

Google Cloud SDK

Google Cloud SDK에는 App Engine, Compute Engine, Cloud Storage, Cloud SQL, BigQuery를 비롯하여 Google Cloud Platform에서 리소스를 만들고 관리할 수 있는 도구와 라이브러리가 포함되어 있습니다.

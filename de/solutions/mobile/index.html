Mobile App mit der
Google Cloud Platform erstellen

Erstellen und hosten Sie das Back-End für beliebige mobile Apps. Dank der automatisch verwalteten Infrastruktur können Sie sich ganz auf Ihre App konzentrieren. Anschließend können Sie eine unbegrenzte Skalierung auf Millionen von Nutzern durchführen.

Jetzt testen oder Vertrieb kontaktieren

App in 300 Sekunden bereitstellen

Erfahren Sie, wie einfach Sie ein mobiles Back-End entwickeln können, indem Sie benutzerdefinierten Code mit Google Cloud Endpoints verwenden. Automatisch erzeugte, stark typisierte Client-Bibliotheken für Android, iOS und JavaScript machen das Aufrufen von Back-End-APIs ganz einfach.

Vorteile der Erstellung mobiler Apps auf der Google Cloud Platform:

Automatische Skalierung und Lastenausgleich

Sie brauchen nur Ihren App-Code zu schreiben – wir erledigen den Rest. Egal, wie viele Nutzer Sie haben, App Engine führt eine Skalierung entsprechend Ihren Anforderungen durch.

Statische Inhalte bereitstellen

App Engine stellt statische Dateien aus dafür vorgesehenen Servern und Caches bereit, die sich von den Anwendungsservern unterscheiden.

Datenverarbeitung

App-Protokolle können Einblicke in die App-Nutzung und das Back-End-Verhalten bieten. BigQuery ermöglicht Ihnen eine Ad-hoc-Analyse der Protokolldateien in Echtzeit. Dabei werden Terabyte an Daten in Sekundenschnelle analysiert.

Geografische Umgebungssuche

Nutzen Sie die Search API von App Engine für eine geografische Umgebungssuche.

Push-Benachrichtigungen

Verwenden Sie das Google Cloud-Messaging zum Senden von Daten an Ihre Android-Nutzer und die Sockets API zum Senden von Push-Benachrichtigungen an Ihre iOS-Nutzer.

Datenspeicherung

Mit Google Cloud Storage können Sie über eine einfache REST-basierte Oberfläche Objekte und Dateien speichern, die mehrere Terabyte groß sein können.

Erste Schritte

Stellen Sie eine mobile App bereit. Alles, was Sie dafür benötigen, ist der Code, die Anleitung und die Google Cloud Platform.

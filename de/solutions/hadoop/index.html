Hadoop skaliert schnell auf der Google Cloud Platform

Die Apache Hadoop-Softwarebibliothek ist ein Framework, das die verteilte Verarbeitung großer Datensätze mithilfe einfacher Programmiermodelle in Computerclustern ermöglicht. Sie wurde so entworfen, dass sie von einzelnen Servern bis hin zu Tausenden von Computern skaliert werden kann, die jeweils eine lokale Berechnung und Speicherung ermöglichen. Statt sich bei der Bereitstellung hoher Verfügbarkeit auf Hardware zu verlassen, wurde die Bibliothek so entworfen, dass Fehler auf Anwendungsebene erkannt und behandelt werden. Dadurch wird ein Dienst mit hoher Verfügbarkeit auf der Grundlage eines Clusters von Computern geboten, von denen einzelne ausfallen können.

"Hadoop auf Google Compute Engine hat die Zuverlässigkeit unserer MapReduce-Aufträge im Vergleich zu früheren Lösungen um über 50 % gesteigert. MapR Technologies

MapR hat mithilfe von Google Compute Engine einen neuen MinuteSort-Weltrekord aufgestellt. In 60 Sekunden wurden 15 Milliarden 100-Byte-Datensätze sortiert, was 1,5 Billionen Byte entspricht. Die Benchmark, die auch als Weltmeisterschaft der Datensortierung bezeichnet wird, hat gezeigt, wie schnell eine Datensortierung möglich ist, die auf Laufwerken anfängt und endet. Während der frühere MinuteSort-Rekord mit benutzerdefinierter Hardware erreicht wurde, hat MapR den neuen Rekord mithilfe von Google Compute Engine, Hadoop MapReduce und MapR Distribution for Apache Hadoop erreicht.

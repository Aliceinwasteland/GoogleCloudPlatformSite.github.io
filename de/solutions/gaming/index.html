Lösungen

Große Mediendateien bereitstellen

Nutzer erwarten zunehmend Bilder in höherer Auflösung und die Möglichkeit von Downloads beliebiger Größe. Sie erreichen, dass Ihre Nutzer wiederkommen und Ihre App weiter nutzen, indem Sie stetig neue Inhalte anbieten. Google Cloud Storage bietet eine einfache REST-basierte Oberfläche zum Speichern großer Dateien in der Cloud. Es nutzt dasselbe Hochgeschwindigkeitsnetzwerk, das auch den Google-Anwendungen zugrunde liegt.

Nutzerinteraktionen analysieren

Sie können den Wert Ihres Spiels nur dann richtig einschätzen, wenn Sie verstehen, wie Nutzer sich verhalten und wie Ihre App bei ihnen abschneidet. Dazu brauchen Sie jedoch mehr als nur Protokolle – Sie brauchen leistungsfähige Analysetools. Mit Google BigQuery können Sie für Ihre Apps dieselbe Technologie nutzen, die wir bei Google für diese Aufgaben verwenden. So können Sie in Echtzeit Ad-hoc-Analysen von vielen Terabyte an Daten durchführen und in Ihren Dashboards speichern. Sie können auch MapReduce auf App Engine verwenden, um große Datensätze parallel für die Nicht-Echtzeit-Analyse zu verarbeiten.

Spielstatus auf mehreren Geräten synchronisieren

Speichern Sie den Spielstatus im App Engine Datastore oder in Google Cloud Storage. Dadurch ist er auf allen Geräten Ihrer Nutzer verfügbar und Sie selbst können daraus nützliche Informationen gewinnen.

Einfache Integration in Mobilgeräte

Verwenden Sie Google Cloud Endpoints zur automatischen Bereitstellung Ihrer Serverlogik als Web-APIs und zum Erzeugen stark typisierter Client-Bibliotheken für Android, iOS und JavaScript für Standard-Web-Apps.

Unterschiedliche Spiele auf der Google-Plattform erstellen

Die Google Cloud Platform ermöglicht Ihnen das Erstellen von Spielen für mobile Plattformen, einschließlich Android und iOS, sowie für Webplattformen wie Facebook und Chrome OS. Sie können verschiedene Arten von Spielen erstellen, von rundenbasierten Spielen bis hin zu Spielen für soziale Netzwerke.

Spiele mit Google verbreiten und monetarisieren

Monetarisieren Sie Ihr Spiel mittels Google AdSense und mit von Google unterstützen In-App-Zahlungen. AdSense ermöglicht Ihnen das Einblenden von Anzeigen mit Targeting, die zum Erscheinungsbild Ihres Spiels passen. Über In-App-Zahlungen können Sie virtuelle Waren für eine Pauschalgebühr von 5 % verkaufen. Vertreiben Sie Ihre Webspiele über den Chrome Web Store und Ihre mobilen Spiele bei Google Play.

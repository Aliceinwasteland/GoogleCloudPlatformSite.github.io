Cloud DNS — Google Cloud Platform

Cloud DNS

Zuverlässige und stabile DNS-Bereitstellung mit geringer Latenzzeit über das weltweite Google-Netzwerk von Anycast-DNS-Servern. Erstellen Sie DNS-Datensätze mithilfe einer benutzerfreundlichen Befehlszeilenschnittstelle oder verwenden Sie für die Programmierung eine umfassende RESTful API, um den Dienst an Ihre spezifischen Anforderungen anzupassen.

Jetzt testen

Funktionen

Leistungsstarker und zuverlässiger DNS-Dienst

Nutzen Sie die Infrastruktur von Google für eine umfangreiche DNS-Bereitstellung in Produktionsqualität. Ihre Nutzer erhalten über unser Netzwerk von Anycast-Nameservern weltweit zuverlässigen Zugriff mit geringen Latenzzeiten auf die Google-Infrastruktur.

Nutzerfreundlich und anpassbar

Verwalten Sie Ihre Datensätze über unsere nutzerfreundliche, skriptfähige Befehlszeilenschnittstelle. Oder erstellen Sie mit der umfassenden RESTful API Ihre eigene benutzerdefinierte DNS-Schnittstelle.

Datensätze für alle Dienste verwalten

Verknüpfen Sie DNS-Datensätze für Google-Plattformdienste wie Compute Engine, App Engine oder Google Cloud Storage.

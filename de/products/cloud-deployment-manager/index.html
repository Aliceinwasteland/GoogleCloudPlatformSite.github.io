Cloud Deployment Manager

Cloud Deployment Manager allows developers to easily design, share, deploy and manage complex Cloud Platform solutions using simple, declerative templates. From simple web servers to complex highly available clusters, Deployment Manager allows teams to spend less time managing, and more time building.

Funktionen

Einfache, wiederholbare Bereitstellung in der Cloud

Mithilfe der deklarativen, parametrierbaren Vorlagen des Deployment Managers können Cloud Platform-Abläufe einfach und wiederholt durchgeführt werden. Die operativen Teams müssen weder den Status ihrer Infrastruktur noch Änderungen verwalten.

Google wacht über Ihre Infrastruktur

Mit dem Deployment Manager kann der Status Ihrer Ressourcen sowohl während als auch nach der Bereitstellung einer Vorlage aktiv überwacht werden. Dabei können viele Arten von Cloud Platform-Ressourcen analysiert werden. Dadurch entfällt die Notwendigkeit, viele APIs miteinander zu verknüpfen.

Einfache Tools und umfassende Integration

Verwenden Sie unsere einfache, skriptfähige Befehlszeilenschnittstelle, um Ihre Bereitstellungen zu verwalten. Oder nutzen Sie eine OAuth 2.0-kompatible RESTful API mit vollem Funktionsumfang für eine umfassende Integration in Ihre eigene Toolkette und Ihre Arbeitsabläufe.

Nutzen Sie die Google-Algorithmen für maschinelles Lernen zum Analysieren von Daten und zum Vorhersagen zukünftiger Ergebnisse mithilfe einer vertrauten RESTful-Benutzeroberfläche.

Funktionen

Eigene Daten nutzen

Sagen Sie anhand von bisherigen Daten zukünftige Trends voraus. Verwenden Sie die Prediction API, um Nachrichten weiterzuleiten, Spam zu erkennen und Nutzern Produkte zu empfehlen. Die leistungsfähige Infrastruktur von Google analysiert die Daten für Sie.

Schnell und zuverlässig

Ihre Daten werden mittels Cloud Storage in mehreren Rechenzentren repliziert. Die meisten Vorhersageabfragen dauern weniger als 200 ms. Auch eine noch höhere Leistung ist möglich. Tipps zur Leistung finden Sie in der Dokumentation.

Cloud-Integration

Google-Dienste sind für die Zusammenarbeit entworfen. Verwenden Sie Ergebnisse aus Google BigQuery oder rufen Sie Datensätze von bis zu 2,5 GB aus Cloud Storage ab, um ein Vorhersagemodell zu erstellen. Verwenden Sie die Prediction API direkt über Google App Engine.

Leistungsfähige Entwicklungstools

Die Prediction API kann in App Engine integriert werden und die RESTful API ist über Bibliotheken für viele gängige Sprachen wie z. B. Python, JavaScript und .NET verfügbar. Die vollständige Liste finden Sie in der Dokumentation.

Flexible Preisgestaltung

In den ersten sechs Monaten ist eine begrenzte Nutzung kostenlos. Darüber hinaus beinhalten die Preise für die Prediction API ein SLA (Service Level Agreement) mit einer Verfügbarkeit von 99,9 %. Wie bei anderen Cloud-Produkten zahlen Sie nur für die tatsächliche Nutzung.

Preise

Kostenloses Kontingent

Im Rahmen der folgenden Nutzungsbegrenzungen pro Google Cloud Console-Projekt ist die Verwendung in den ersten sechs Monaten kostenlos. Dieses kostenlose Kontingent gilt auch bei aktivierter Abrechnung bis zum Ende der sechs Monate.

  • Nutzungsbegrenzungen:
    • Vorhersagen: 100 Vorhersagen/Tag
    • Vorhersagen für gehostete Modelle: Alle gehosteten Modelle unterliegen einer Nutzungsbegrenzung von 100 Vorhersagen/Tag pro Nutzer.
    • Schulung: 5 MB geschult/Tag
    • Streamingupdates: 100 Streamingupdates/Tag
  • Begrenzung für gesamten Zyklus: 20.000 Vorhersagen
  • Verfallsdatum: Das kostenlose Kontingent verfällt sechs Monate nach dem Aktivieren von Google Prediction für Ihr Projekt in der Google Cloud Console.

Hinweis: Einige Nutzungsszenarien der Prediction API hängen von Google Cloud Storage ab, für das andere Preise als für die Prediction API gelten.

Bezahlte Nutzung

Für die bezahlte Nutzung gilt ein SLA mit einer Verfügbarkeit von 99,9 %. Weitere Informationen zum SLA

Überwachen Sie Ihre Nutzung in der Google Cloud Console. Eine Nutzung jenseits des kostenlosen Kontingents ist nur möglich, wenn Sie für das Projekt die Abrechnung aktivieren. Sobald das kostenlose Kontingent ausgeschöpft ist, gelten die folgenden Preise und Begrenzungen:

  • Grundgebühr: 10 $/Monat pro Google Cloud Console-Projekt
  • Nutzungsgebühren:
    • Vorhersage:
      • 10.000 Vorhersagen/Monat: 0,00 $
      • Über 10.001 Vorhersagen/Monat: 0,50 $/1.000 Vorhersagen nach den ersten 10.000
    • Schulung:
      • 0,002 $/MB bulk-geschult (maximale Größe jedes Datensatzes: 2,5 GB)
      • 0–10.000 Streamingupdates: 0,00 $
      • Über 10.001 Streamingupdates: 0,05 $/1.000 Updates nach den ersten 10.000
  • Nutzungsbegrenzungen:
    • Wenn Sie vorhaben, mehr als 40.000 Vorhersagen pro Tag durchzuführen, nehmen Sie bitte Kontakt mit uns auf.
    • Für Google Prediction gilt eine Standardbegrenzung von 2.000.000 Vorhersagen/Tag pro Projekt für Ihre eigenen Modelle.
    • Für alle gehosteten Modelle gilt eine Nutzungsbegrenzung von 100 Vorhersagen/Tag pro Nutzer für sämtliche Vorführmodelle. Entwickler können für von ihnen erstellte Modelle höhere bezahlte Nutzungsbegrenzungen festlegen. Weitere Details finden Sie in der Dokumentation der jeweiligen Modelle.

Wichtig: Diese Gebühren beinhalten keine Google Cloud Storage-Gebühren, die möglicherweise für die Speicherung von Schulungsdaten anfallen. In der Preisübersicht für Google Cloud Storage finden Sie Informationen zu zusätzlichen Kosten, die unter Umständen anfallen können.

Ressourcen

  • Was ist die Google Prediction API?

    Die cloud-basierten Tools von Google für maschinelles Lernen können dabei helfen, Ihre Daten zu analysieren und zu Ihren Anwendungen beispielsweise die folgenden Funktionen hinzuzufügen: Kundenstimmungsanalyse, Spamerkennung, Entscheidungen zur Weiterleitung von Nachrichten, Analyse von Upselling-Gelegenheiten sowie Dokument- und E-Mail-Klassifizierung.

  • Vorhersage von Käufen

    Stellen Sie sich eine Website vor, über die Bier, Wein und Käse verkauft wird. Sie möchten anhand der bisherigen Käufe eines Besuchers vorhersagen, ob er sich für Wein interessiert. In dieser Situation könnten wir Schulungsdaten mit drei Angaben erstellen: wie oft der Kunde bereits Dessertkäse gekauft hat, ob der Kunde jemals Wein gekauft hat und wie oft der Kunde bereits Bier gekauft hat.

  • Stimmungsanalysemodell erstellen

    In diesem Dokument wird das Erstellen eines grundlegenden Stimmungsanalysemodells mithilfe der Google Prediction API erläutert. Ein Stimmungsanalysemodell dient dazu, einen Text-String zu analysieren und mit einem der von Ihnen bereitgestellten Labels zu klassifizieren. Sie könnten zum Beispiel einen Tweet daraufhin analysieren, ob er positiv oder negativ ist, oder eine E-Mail daraufhin, ob sie Freude, Frustration oder Trauer ausdrückt.

Products — Google Cloud Platform

Produkte

Führen Sie Ihre Apps mit denselben Technologien und Tools aus, die auch von Google verwendet werden.Die Cloud Platform bietet Ihnen die erforderlichen Bausteine und Dienste für eine schnelle Entwicklung.

Computing

Compute Engine

Compute Engine ist der IaaS-Dienst von Google. Bewältigen Sie hohe Arbeitslasten auf virtuellen Maschinen (VMs), die auf der Infrastruktur von Google gehostet sind. Wählen Sie eine VM und profitieren Sie von der Leistungsfähigkeit des globalen Google-Glasfasernetzwerks. Dank Abrechnung pro Minute zahlen Sie nur für die tatsächliche Nutzung.

  • Fallstudien
  • Preise
  • Dokumentation

App Engine

App Engine ist der PaaS-Dienst von Google. Entwickeln Sie Ihre Apps mithilfe der integrierten Dienste besonders produktiv. Stellen Sie eine vollständig verwaltete Plattform bereit und überlassen Sie Google die Verwaltung des Pagers. Laden Sie kostenlos das SDK herunter und beginnen Sie sofort mit der Entwicklung (keine Kreditkarte erforderlich).

Speicherung

Cloud SQL

Speichern und verwalten Sie Daten mithilfe einer vollständig verwalteten, relationalen MySQL-Datenbank. Google übernimmt Replikation, Patch-Verwaltung und Datenbankverwaltung und sorgt für hohe Verfügbarkeit und Leistung.

  • Funktionen
  • Fallstudien
  • Preise
  • Dokumentation

Cloud Storage

Nutzen Sie einen zuverlässigen Objektspeicherdienst mit hoher Verfügbarkeit. Dank globalem Edge-Caching können Ihre Nutzer überall schnell auf die Daten Ihrer App zugreifen. Google übernimmt die Versionsverwaltung, garantiert ein starkes SLA und bietet eine einfache API für die programmatische Verwaltung Ihrer Daten.

  • Funktionen
  • Fallstudien
  • Preise
  • Dokumentation

Cloud Datastore

Cloud Datastore bietet eine verwaltete schemalose NoSQL-Datenbank zum Speichern nicht relationaler Daten. Cloud Datastore sorgt für eine automatische Skalierung entsprechend Ihrem Bedarf und unterstützt Transaktionen und leistungsfähige, SQL-ähnliche Abfragen.

  • Preise
  • Dokumentation

Big Data

BigQuery

Analysieren Sie Big Data in der Cloud mittels BigQuery. Führen Sie in Sekundenschnelle SQL-ähnliche Abfragen in mehrere Terabyte umfassenden Datensätzen aus. BigQuery ist skalierbar und benutzerfreundlich und bietet Ihnen Echtzeitinformationen zu Ihren Daten.

Dienste

Cloud DNS

Zuverlässige und stabile DNS-Bereitstellung mit geringer Latenzzeit über das weltweite Google-Netzwerk von Anycast-DNS-Servern. Erstellen Sie DNS-Datensätze mithilfe einer benutzerfreundlichen Befehlszeilenschnittstelle oder verwenden Sie für die Programmierung eine umfassende RESTful API, um den Dienst an Ihre spezifischen Anforderungen anzupassen.

  • Funktionen
  • Preise
  • Dokumentation

Cloud Endpoints

Entwickeln Sie mit Ihrem Code RESTful-Dienste und stellen Sie sie für Android-, iOS- und JavaScript-Clients bereit. Erzeugen Sie automatisch Client-Bibliotheken, um das Anschließen des Front-End zu vereinfachen. Die integrierte kritische Infrastruktur bietet DoS-Schutz (Denial-of-Service), Unterstützung für OAuth 2.0 und Verwaltung von Client-Schlüsseln.

Verwaltung

Cloud Deployment Manager

Mit dem Cloud Deployment Manager können Entwickler komplexe Cloud Platform-Lösungen anhand einfacher, deklarativer Vorlagen problemlos erstellen, teilen, bereitstellen und verwalten. Ob einfache Webserver oder komplexe hochverfügbare Cluster: Mit dem Deployment Manager sparen Teams Zeit bei der Verwaltung, die sie so für die Entwicklung gewinnen.

Entwicklertools

Google Cloud SDK

Das Google Cloud SDK bietet Ihnen Tools und Bibliotheken für die Erstellung und Verwaltung von Ressourcen auf der Google Cloud Platform, einschließlich App Engine, Compute Engine, Cloud Storage, Cloud SQL und BigQuery.

Informationen zum Google Cloud SDK

Push-to-Deploy

Verwenden Sie Git zur automatischen Bereitstellung Ihrer App in App Engine. Sie können mit einem Repository-Push eine neue Version veröffentlichen und erhalten Zugriff auf alle Funktionen von Git, einschließlich eines Commit-Verlaufs.

Informationen zu Push-to-Deploy

Cloud Playground

Führen Sie Cloud Platform-Dienste wie App Engine, Cloud Storage und Cloud SQL direkt über Ihren Browser aus, um sie besonders schnell zu testen.

Zu Cloud Playground

Android Studio

Fügen Sie die Cloud Platform als Back-End direkt über die Android Studio-IDE zu Ihrer App hinzu. Google Cloud Endpoints bietet eine API für Ihr Web-Back-End und aktualisiert sie automatisch auf allen Ihren Front-End-Plattformen.

Google-Plug-in für Eclipse

Es bietet Tools, API-Unterstützung und eine einfache Bereitstellung für App Engine-Java-Entwickler.

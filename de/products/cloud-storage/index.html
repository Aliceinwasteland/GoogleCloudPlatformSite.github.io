Cloud Storage

Nutzen Sie einen zuverlässigen Objektspeicherdienst mit hoher Verfügbarkeit. Dank globalem Edge-Caching können Ihre Nutzer überall schnell auf die Daten Ihrer App zugreifen.

Funktionen

Sicher und geschützt

Ihre Daten sind durch redundante Speicherung an mehreren physischen Standorten geschützt. OAuth und detaillierte Zugriffssteuerung sorgen für hohe, konfigurierbare Sicherheit.

Attraktive, flexible Preisgestaltung

Wählen Sie eine Speicherlösung, die zur Art Ihrer Daten passt, ob Standardspeicherung oder dauerhafte Speicherung mit verringerter Verfügbarkeit. Wie bei anderen Cloud Platform-Diensten zahlen Sie nur für die tatsächliche Nutzung.

Objektspeicherung mit einer API mit vollem Funktionsumfang

Cloud Storage ist über eine einfache API zugänglich. Zusätzlich sorgen erweiterte Funktionen für mehr Flexibilität und Leistung. Die Verwaltung von Objektversionen, parallele Uploads und CRC-basierte Integritätsprüfung ermöglichen Entwicklern die Erstellung komplexer, stabiler Apps. Sie können mittels XML oder JSON direkt auf die API zugreifen oder Bibliotheken für verschiedene gängige Programmiersprachen verwenden.

Flexibler Zugriff

Sie können statische Objekte direkt aus Cloud Storage bereitstellen oder können Cloud Storage als Datenspeicher für andere Google-Produkte verwenden. Sie können über BigQuery, App Engine oder andere Cloud Platform-Dienste auf Ihren Speicher zugreifen. Ihre Daten sind über eine authentifizierte RESTful API zugänglich, die hohe Geschwindigkeit, Flexibilität und von OAuth unterstützte Sicherheit bietet.

Ressourcen

  • Überblick über Google Cloud Storage

    Google Cloud Storage ermöglicht weltweit rund um die Uhr das Speichern und Abrufen beliebiger Datenmengen. Es bietet eine einfache Programmierschnittstelle, über die Entwickler die zuverlässige und schnelle Netzwerkinfrastruktur von Google nutzen und dadurch auf sichere und kostengünstige Weise Datenoperationen durchführen können. Wenn eine Erweiterung nötig wird, profitieren Entwickler von der Skalierbarkeit, die die Google-Infrastruktur bietet.

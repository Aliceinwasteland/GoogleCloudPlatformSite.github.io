Translate API — Google Cloud Platform

Translate API

Führen Sie durch die die Integration von Google Translate innerhalb der App eine schnelle, dynamische Übersetzung zwischen Tausenden verfügbaren Sprachpaaren durch.

Jetzt testen

Funktionen

Dynamischer Zugriff auf Sprachen

Google Translate API unterstützt Dutzende von Sprachen. Ermitteln Sie die Sprache eines Dokuments und übersetzen Sie es mithilfe einer RESTful API.

Zugriff mit Google API

Auf die Translate API kann über die standardmäßige Google API zugegriffen werden. Weitere Informationen finden Sie unter Beispielcode und Bibliotheken für zehn verschiedene Programmiersprachen, einschließlich Python, Objective C und Ruby.

Attraktive, einfache Preisgestaltung

Die Translate API-Preise werden anhand dessen berechnet, wie viele Millionen von Zeichen gesendet wurden. Unsere Preise sind leicht nachzuvollziehen und sehr günstig.

Preise

Die Google Translate API-Preise sind nutzungsbasiert. Die Nutzung wird in Millionen (Mio.) von Zeichen berechnet. 1 Mio. Zeichen entsprechen dabei 106 Zeichen. Weitere Informationen finden Sie in den häufig gestellten Fragen zur Preisgestaltung.

Wichtig: Google Translate API Version 2 erfordert für alle Konten Zahlungsinformationen, bevor Sie mit der Nutzung des Diensts beginnen können. In der Anleitung unten finden Sie Informationen zum Aktivieren der Abrechnung.

  • Nutzungsgebühren:
    • Übersetzung:
      • 20 $ pro 1 Mio. Zeichen. Dabei richten sich die Gebühren nach der tatsächlich bereitgestellten Anzahl von Zeichen. Wenn Sie zum Beispiel 500.000 Zeichen übersetzen lassen, werden Ihnen 10 $ in Rechnung gestellt.
    • Spracherkennung:
      • 20 $ pro 1 Mio. Zeichen. Dabei richten sich die Gebühren nach der tatsächlich bereitgestellten Anzahl von Zeichen.
  • Nutzungsbegrenzungen:
    • Für die Google Translate API gilt eine Standardbegrenzung von 2 Mio. Zeichen pro Tag. Sie können diese Begrenzung in der Cloud Console anhand der unten stehenden Anleitung auf bis zu 50 Mio. Zeichen pro Tag erweitern.
    • Wenn Sie mehr als 50 Mio. Zeichen pro Tag übersetzen müssen, nehmen Sie bitte Kontakt mit uns auf.

Gehen Sie zum Aktivieren der Abrechnung für Ihr Projekt folgendermaßen vor:

  1. Gehen Sie zur Google Cloud Console.
  2. Wählen Sie ein Projekt aus.
  3. Wählen Sie in der linken Seitenleiste die Option Abrechnung aus.
  4. Wählen Sie oben auf der Seite die Option Abrechnung aktivieren aus.
  5. Füllen Sie das Formular aus.

Wenn Sie die Nutzungsbegrenzungen für Ihr Projekt aufrufen oder ändern oder eine Erhöhung Ihrer Gesamtbegrenzung beantragen möchten, gehen Sie wie folgt vor:

  1. Gehen Sie zur Google Cloud Console.
  2. Wählen Sie ein Projekt aus.
  3. Wählen Sie in der linken Seitenleiste die Option APIs & Auth. und anschließend eine API aus.
  4. Klicken Sie auf der API-Infoseite in der Nähe des API-Namens auf den Link Kontingent.

BigQuery

Analysieren Sie Big Data in der Cloud mittels BigQuery. Führen Sie in Sekundenschnelle SQL-ähnliche Abfragen in mehrere Terabyte umfassenden Datensätzen aus. BigQuery ist skalierbar und benutzerfreundlich und bietet Ihnen Echtzeitinformationen zu Ihren Daten.

Funktionen

Alles im Hintergrund

Ihre Abfragen können asynchron im Hintergrund ausgeführt werden und können abgefragt werden, um den Status zu bestimmen. Greifen Sie auf den Verlauf Ihrer Aufträge und Abfragen sowie auf den Rest Ihrer Cloud Platform-Ressourcen in der Google Cloud Console zu.

Einfacher Datenimport

Laden Sie Ihre Daten im Bulk mittels Google Cloud Storage oder streamen Sie sie in Paketen von bis zu 1.000 Zeilen pro Sekunde. Sie können in Echtzeit umfassende Protokolle erstellen und Transaktionen erfassen sowie für die Analyse schnell und einfach auf Ihre Daten zugreifen.

Kostengünstige Big Data

Wenn wir von "Big Data" sprechen, meinen wir das auch. Die ersten 100 GB verarbeiteter Daten pro Monat sind kostenlos. Wir speichern Tausende von Terabyte für Sie und fragen sie für Sie ab. Unsere Preisstruktur ist einfach und transparent und Sie erhalten vollständigen Einblick und uneingeschränkte Kontrolle. Sie zahlen nur für den Speicherplatz, den Sie benötigen, und die Abfragen, die Sie ausführen.

Die richtige Benutzeroberfläche

Die separaten Oberflächen für Verwaltung und Entwicklung bieten Ihnen Zugriff auf die Tools, die Sie benötigen. Steuern Sie den Zugriff sowohl auf Projekt- als auch auf Datensatzebene mithilfe der Google APIs Console. Mit BigQuery können Teams jeder Größe so arbeiten, wie sie möchten.

Preise

Bei BigQuery wird eine spaltenbasierte Datenstruktur verwendet. Das heißt, für eine Abfrage werden nur die in jeder Spalte verarbeiteten Daten berechnet, nicht die gesamte Tabelle. Die ersten 100 GB verarbeiteter Daten pro Monat sind kostenlos.

Die Preise gelten ab 1. April 2014.

Ressourcen

  • Was ist BigQuery?

    Das Abfragen umfangreicher Datensätze kann ohne die richtige Hardware und Infrastruktur zeitaufwendig und teuer sein. Google BigQuery löst dieses Problem durch extrem schnelle, SQL-ähnliche Abfragen in Tabellen, für die nur Anfügungen zulässig sind. Dabei wird die Rechenleistung der Google-Infrastruktur genutzt. Verschieben Sie einfach Ihre Daten zu BigQuery und überlassen Sie uns die Schwerarbeit.

  • Daten in BigQuery laden

    Bevor Sie Ihre Daten abfragen können, müssen Sie sie zuerst in BigQuery laden. Sie können die Daten über einen Auftrag im Bulk laden oder Datensätze einzeln streamen.

  • Daten abfragen

    Abfragen werden im SQL-Dialekt von BigQuery geschrieben. BigQuery unterstützt sowohl synchrone als auch asynchrone Abfragemethoden. Beide Methoden werden von einem Auftrag gesteuert, aber bei der synchronen Methode wird ein Zeitlimit angeben, bis zu dem vor der Ausgabe auf den Abschluss des Auftrags gewartet wird.

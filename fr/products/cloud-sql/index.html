Cloud SQL

Stockez et gérez les données à l'aide d'une base de données relationnelle MySQL entièrement gérée. Nous nous chargeons de la réplication, de la gestion des correctifs et de l'administration de la base de données afin d'assurer une disponibilité et des performances optimales.

Essayer maintenant

Fonctionnalités

Profitez sans difficulté de bases de données MySQL déployées dans le cloud. Avec Google Cloud Platform, vous disposez de bases de données puissantes et rapides. Vous n'êtes jamais à court d'espace, et vous bénéficiez d'un stockage redondant et fiable pour vos applications.

Infrastructure connue

Développez des applications et déployez-les plus rapidement dans le cloud. Cloud SQL fournit des bases de données MySQL standards, autrement dit les bases de données Open Source les plus populaires au monde. Les instances disponibles disposent d'une mémoire RAM pouvant atteindre 16 Go et d'un espace de stockage de 100 Go.

Facturation souple

Prenez un bon départ sans vous ruiner grâce à notre option de facturation à l'utilisation. Si votre base de données n'est que faiblement ou occasionnellement sollicitée, faites des économies en ne payant que le temps réel d'accès aux données. Pour les instances soumises à des charges plus intensives, maîtrisez vos coûts grâce à l'option forfaitaire.

Sécurité, disponibilité et durabilité

Your data is replicated in many geographic locations as standard, and failover between them is handled automatically. This means your data is safe and your database is available even in the event of a major failure. We also manage your backups, making it easy for you to restore when needed, including point-in-time recovery.

Migration facilitée et libre choix du fournisseur

Des outils et des connexions standards tels que mysqldump, MySQL Wire Protocol et JDBC facilitent la migration vers (mais aussi depuis) Google Cloud Platform. Vous ne dépendez pas d'un fournisseur particulier.

Contrôle

It's easy to manage and access your instances through a web Console or a command-line interface.

Service entièrement géré

Plus besoin de s'occuper de la réplication, de la gestion des correctifs ni de celle des données. Nous nous en chargeons.

Tarifs

Nous proposons deux programmes de facturation pour Cloud SQL :

  • Pour les développeurs qui enregistrent un trafic plus important, nous proposons des forfaits avec remise, et nous les aidons à prévoir leurs coûts.
  • Pour les développeurs d'applications légères, nous proposons un programme de facturation flexible en fonction de l'utilisation. Vous ne payez que le temps réel d'accès aux données. Démarrez avec une base de données MySQL hébergée dans le cloud pour environ 1 USD par mois.

  • À propos de Google Cloud SQL

    Google Cloud SQL est une base de données MySQL hébergée dans le cloud de Google. Hormis quelques fonctionnalités indisponibles, elle offre les mêmes possibilités et fonctionnalités que MySQL, ainsi que quelques fonctionnalités supplémentaires. Simple d'emploi, Google Cloud SQL ne nécessite aucune installation de logiciel ni maintenance. Elle convient particulièrement aux applications de petite et moyenne taille.

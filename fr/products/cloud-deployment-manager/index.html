Cloud Deployment Manager

Cloud Deployment Manager allows developers to easily design, share, deploy and manage complex Cloud Platform solutions using simple, declerative templates. From simple web servers to complex highly available clusters, Deployment Manager allows teams to spend less time managing, and more time building.

Fonctionnalités

Déploiements simples et reproductibles dans le Cloud

Grâce aux modèles déclaratifs à paramétrer de Deployment Manager, les opérations sur Cloud Platform sont simples et reproductibles, ce qui permet aux équipes chargées des opérations de se dégager de la gestion de l'état de leur infrastructure ou du suivi des changements.

Laissez à Google le soin de s'occuper de votre infrastructure

Deployment Manager peut surveiller activement l'état de vos ressources à la fois pendant et après le déploiement d'un modèle. Il peut traiter de nombreux types de ressources Cloud Platform, éliminant ainsi la nécessité de mettre en œuvre un grand nombre d'API.

Des outils simples et une intégration poussée

Utilisez notre interface de ligne de commande d'une grande simplicité permettant de rédiger des script pour assurer la gestion de vos déploiements, ou exploitez une API RESTful dotée de fonctionnalités complètes et conforme au protocole OAuth 2.0 pour effectuer une intégration poussée dans votre propre chaîne d'outils et vos flux de travail.

Translate API — Google Cloud Platform

Translate API

Traduisez rapidement des textes de manière dynamique au sein de votre application grâce à l'intégration avec Google Translate. Des milliers de combinaisons linguistiques sont disponibles.

Essayer maintenant

Fonctionnalités

Accès aux langues de manière dynamique

Google Translate API est compatible avec plusieurs dizaines de langues. Identifiez la langue d'un document, puis traduisez celui-ci à l'aide d'une API RESTful.

Accès via l'API Google

Translate API est accessible via l'API Google standard. Consultez les exemples de code et les bibliothèques de dix langages de programmation, y compris Python, Objective C et Ruby.

Tarification simple et abordable

Les tarifs de Translate API sont calculés par tranche de 1 million de caractères envoyés. Notre barème de prix est facile à comprendre et très abordable.

Tarifs

Les tarifs de Google Translate API sont calculés en fonction de votre consommation. Celle-ci est calculée en millions de caractères (M), où 1 M = 106 caractères. Pour plus d'informations, consultez les questions fréquentes relatives aux tarifs.

Important : Pour utiliser Google Translate API v2, vous devez d'abord vous assurer que tous les comptes comportent des informations de facturation. Pour savoir comment activer la facturation, reportez-vous aux instructions ci-dessous.

  • Frais d'utilisation :
    • Traduction :
      • 20 USD pour 1 M de caractères de texte, au prorata du nombre de caractères effectivement fournis. Par exemple, la traduction de 500 000 caractères est facturée 10 USD.
    • Détection de la langue :
      • 20 USD pour 1 M de caractères de texte, au prorata du nombre de caractères effectivement fournis.
  • Limites d'utilisation :
    • Par défaut, Google Translate API est limité à 2 M de caractères par jour. Vous pouvez augmenter cette limite jusqu'à 50 M de caractères par jour dans Cloud Console en suivant les instructions ci-dessous.
    • Si vous devez traduire plus de 50 M de caractères par jour, veuillez nous contacter.

Pour activer la facturation pour votre projet, procédez comme suit :

  1. Accédez à Google Cloud Console.
  2. Sélectionnez un projet.
  3. Dans la barre latérale située à gauche, sélectionnez Facturation.
  4. En haut de la page, sélectionnez Activer la facturation.
  5. Remplissez le formulaire.

Pour afficher ou modifier les limites d'utilisation de votre projet, ou pour demander une augmentation de la limite totale, procédez comme suit :

  1. Accédez à Google Cloud Console.
  2. Sélectionnez un projet.
  3. Dans la barre latérale située à gauche, sélectionnez API et autorisations, puis sélectionnez une API.
  4. Sur la page d'information de l'API, sélectionnez le lien Quota près du nom de l'API.

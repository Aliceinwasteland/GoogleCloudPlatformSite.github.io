Produits

Exécutez votre application avec les outils et technologies utilisés chez Google.
Avec Cloud Platform, vous utilisez les services dont vous avez besoin pour développer rapidement votre projet.

Calcul

Compute Engine

Compute Engine est l'infrastructure en tant que service (IaaS, Infrastructure-as-a-Service) de Google. Exécutez des charges de travail à grande échelle sur des machines virtuelles hébergées dans l'infrastructure de Google. Choisissez une VM qui correspond à vos besoins, et bénéficiez des performances et de la cohérence du réseau de fibre optique mondial de Google. Grâce à la facturation à la minute, vous ne payez que ce que vous consommez.

App Engine

App Engine est la plate-forme en tant que service (PaaS, Platform-as-a-Service) de Google. Développez facilement votre application grâce à des services intégrés qui augmentent votre productivité. Déployez-la sur une plate-forme entièrement gérée. Nous nous occupons du reste. Téléchargez le kit de développement logiciel SDK (Software Development Kit) et commencez immédiatement à développer votre application gratuitement sans entrer de carte de paiement.

Cloud SQL

Stockez et gérez les données à l'aide d'une base de données relationnelle MySQL entièrement gérée. Nous nous chargeons de la réplication, de la gestion des correctifs et de l'administration de la base de données afin d'assurer une disponibilité et des performances optimales.

Cloud Storage

Profitez d'un service de stockage d'objets durable et hautement disponible. Grâce à la mise en cache globale en périphérie du réseau, les utilisateurs accèdent rapidement aux données de votre application, où qu'ils se trouvent. Nous assurons la gestion des versions et nous garantissons le respect d'un contrat de niveau de service strict. Une API simple vous permet de gérer vos données par programmation.

Cloud Datastore

Cloud Datastore est une base de données gérée NoSQL sans schéma permettant de stocker les données non relationnelles. Elle évolue automatiquement en fonction de vos besoins et gère aussi bien les transactions que les requêtes robustes de type SQL.

Big Data

BigQuery

Analysez le Big Data dans le cloud avec BigQuery. Appliquez des requêtes de type SQL à des ensembles de données de plusieurs téraoctets en quelques secondes seulement. À la fois évolutif et simple d'emploi, BigQuery vous permet d'analyser vos données en temps réel.

Services

Cloud DNS

Service DNS fiable, souple, à faible latence depuis le réseau mondial de Google de serveurs DNS Anycast. Créez des enregistrements DNS avec une interface facile à utiliser en ligne de commande, ou depuis une API RESTful très complète pour personnaliser le service à vos besoins spécifiques.

Cloud Endpoints

À partir de votre code, créez des services RESTful accessibles depuis les clients iOS, Android et JavaScript. Facilitez la connexion frontend en générant automatiquement des bibliothèques clientes. L'infrastructure critique intégrée comprend la protection contre les attaques par déni de service, la compatibilité avec OAuth 2.0 et la gestion des clés des clients.

Management

Cloud Deployment Manager

Cloud Deployment Manager permet aux développeurs de facilement concevoir, partager, déployer et gérer des solutions complexes de plate-forme Cloud avec des modèles déclaratifs simples. Que ce soit des serveurs Web simples ou des clusters à haute disponibilité complexes, Deployment Manager permet aux équipes de passer moins de temps à gérer, et plus de temps à construire.

Outils de Développement

SDK Google Cloud

Le SDK Google Cloud contient des outils et des bibliothèques qui vous permettent de créer et de gérer des ressources sur Google Cloud Platform, y compris App Engine, Compute Engine, Cloud Storage, Cloud SQL et BigQuery.

En savoir plus sur le SDK Google Cloud

Déploiement push

Utilisez Git pour déployer automatiquement votre application sur App Engine. Vous pouvez lancer une nouvelle version grâce à une opération Push vers le référentiel et accéder à toutes les fonctionnalités de Git, y compris l'historique des validations.

En savoir plus sur le déploiement push

Cloud Playground

Testez rapidement des services Cloud Platform comme App Engine, Cloud Storage et Cloud SQL en les exécutant directement dans votre navigateur.

Visiter Cloud Playground

Android Studio

Ajoutez Cloud Platform comme backend de votre application directement depuis l'IDE Android Studio (Integrated Development Environment). Avec Google Cloud Endpoints, bénéficiez d'une API pour votre backend Web et de mises à jour automatiques sur toutes vos plates-formes frontend.

Télécharger Android Studio

Plug-in Google pour Eclipse

Avec le plug-in Google pour Eclipse, les développeurs Java sur App Engine bénéficient d'outils adaptés, de fonctionnalités de gestion des API et d'une grande facilité de déploiement.

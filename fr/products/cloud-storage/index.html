Cloud Storage

Profitez d'un service de stockage d'objets durable et hautement disponible. Grâce à la mise en cache globale en périphérie du réseau, les utilisateurs accèdent rapidement aux données de votre application, où qu'ils se trouvent.

Essayer maintenant

Fonctionnalités

Sécurité

Vos données sont protégées grâce à un stockage redondant sur plusieurs sites géographiques différents. Le protocole OAuth et les contrôles d'accès granulaires vous assurent une sécurité fiable et configurable.

Tarification souple et compétitive

Selon la nature de vos données, choisissez une solution de stockage standard ou à disponibilité limitée durable (DRA, Durable Reduced Availability). Comme pour les autres services Cloud Platform, vous ne payez que ce que vous consommez.

Stockage d'objets avec une API riche en fonctionnalités

Pour accéder à Cloud Storage, il vous suffit d'une API simple, mais des fonctionnalités avancées vous offrent encore plus de flexibilité et de puissance. La gestion des versions d'objets, les importations en parallèle et les contrôles d'intégrité via CRC facilitent le développement d'applications robustes et sophistiquées. Vous pouvez accéder directement à l'API si vous utilisez les formats XML ou JSON, tandis que des bibliothèques sont disponibles pour les langages de programmation les plus courants.

Accès flexible

Vous pouvez diffuser des objets statiques directement à partir de Cloud Storage, ou l'utiliser comme un magasin de données pour d'autres produits Google. L'accès au stockage est possible depuis BigQuery, App Engine ou d'autres services Cloud Platform. Vos données sont accessibles par le biais d'une API RESTful authentifiée qui associe rapidité, flexibilité et sécurité basée sur OAuth.

Options de stockage

Outre l'option de stockage standard, nous proposons une option de stockage à disponibilité limitée durable (DRA, Durable Reduced Availability). Il s'agit d'un espace de stockage durable à moindre coût, mais dont la disponibilité attendue est légèrement inférieure. En savoir plus

Les prix entrent en vigueur à compter du 1er avril 2014.

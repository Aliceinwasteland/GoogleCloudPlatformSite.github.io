Mobile App Solutions — Google Cloud Platform

Développez votre application mobile
avec Google Cloud Platform

Créez et hébergez le backend de toute application mobile. L'infrastructure gérée automatiquement vous permet de vous concentrer sur votre application. Profitez d'une évolutivité illimitée pour accueillir des millions d'utilisateurs.

Essayer maintenant ou contacter le service commercial

Déploiement d'application en 300 secondes

Google Cloud Endpoints vous permet de développer facilement le backend d'une application mobile à l'aide de code personnalisé. Les appels d'API backend sont facilités par la génération automatique de bibliothèques clientes fortement typées pour iOS, Android et JavaScript.

Avantages du développement d'applications mobiles sur Google Cloud Platform

Équilibrage des charges et évolutivité automatiques

Il vous suffit de rédiger le code de votre application. Nous nous chargeons du reste. Quel que soit le nombre d'utilisateurs, App Engine évolue en fonction de vos besoins.

Diffusion de contenu statique

App Engine diffuse les fichiers statiques à partir de caches et de serveurs dédiés, distincts des serveurs d'applications.

Traitement des données

Les journaux des applications fournissent des informations sur l'utilisation de celles-ci ainsi que sur le fonctionnement du backend. BigQuery vous permet de réaliser une analyse ad hoc en temps réel de ces fichiers journaux en traitant plusieurs téraoctets de données en quelques secondes.

Recherche à proximité

Effectuez des recherches à proximité grâce à l'API Search d'App Engine.

Notifications push

Envoyez des données aux utilisateurs d'Android à l'aide de Google Cloud Messaging, et des notifications push aux utilisateurs d'iOS à l'aide de l'API Sockets.

Stockage des données

Google Cloud Storage vous permet de stocker des fichiers et des objets d'une taille pouvant atteindre plusieurs téraoctets à l'aide d'une interface REST simple.

Premiers pas

Pour déployer une application mobile, il vous suffit du code, du didacticiel et de Google Cloud Platform.
